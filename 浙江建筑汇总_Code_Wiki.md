## 2. 世界文化遗产

### 2.1 杭州西湖文化景观

| 属性 | 内容 |
|---|---|
| 列入时间 | 2011年 |
| 位置 | 杭州市西湖区 |
| 核心价值 | "一山、二塔、三岛、三堤、五湖"的东方美学典范 |
| 特色 | 免费开放的"西湖模式"，环湖公园景点和博物馆全部免费 |

西湖是东方美学最具代表性的诗意山水，苏东坡"欲把西湖比西子，淡妆浓抹总相宜"传诵千年。景区内有雷峰塔、保俶塔、三潭印月、苏堤、白堤等经典建筑与景观。

### 2.2 中国大运河（杭州段）

| 属性 | 内容 |
|---|---|
| 列入时间 | 2014年 |
| 位置 | 杭州市拱墅区等 |
| 核心价值 | 世界上最长的人工运河，黄金水道流淌千年 |
| 代表遗产点 | 拱宸桥、富义仓、洋关旧址、通益公纱厂旧址等 |

**拱宸桥**：京杭大运河南端的终点标志，始建于1631年，桥身长92.1米，桥高约16米，是杭城最高最长的石拱桥。2014年入选世界文化遗产点。

### 2.3 良渚古城遗址

| 属性 | 内容 |
|---|---|
| 列入时间 | 2019年 |
| 位置 | 杭州市余杭区良渚街道和瓶窑镇 |
| 核心价值 | 实证中华五千年文明史的圣地 |
| 时代 | 新石器时代晚期（约公元前3300—前2300年） |

良渚古城遗址代表了中华文明起源阶段稻作农业的最高成就，由宫殿区、内城、外城组成，拥有复杂的水利系统，是人类文明发展史上具有杰出代表性的东亚地区史前大型聚落遗址。

## 4. 浙江民居流派

### 4.1 浙北（杭嘉湖平原）水乡派

| 属性 | 内容 |
|---|---|
| 核心特征 | 粉墙黛瓦，轻盈通透 |
| 结构 | 穿斗式或穿斗与抬梁混合式木构架 |
| 布局 | 紧凑，庭院小巧，注重临水空间（水埠头） |
| 马头墙 | 形式多样，兼具防火与装饰功能 |
| 代表 | 乌镇民居、西塘民居、南浔古镇 |

### 4.2 浙中（金衢盆地）工匠派（婺派）

| 属性 | 内容 |
|---|---|
| 核心特征 | 木雕装饰繁复精美，尤以东阳木雕为最 |
| 结构 | 规整合院式（三合院、四合院），规模宏大，厅堂高敞 |
| "十三间头" | 婺派建筑基本单元，三合院形式，融合居住、祭祀与生产 |
| "三雕"艺术 | 砖雕门楼、木雕梁架、石雕柱础 |
| 代表 | 东阳卢宅、兰溪诸葛村、浦江郑义门 |

**婺派 vs 徽派区别**：

| 维度 | 婺派 | 徽派 |
|---|---|---|
| 核心技艺 | 突出木构技艺与空间实用 | 重砖雕粉墙 |
| 布局 | "十三间头"兼顾居住与农耕 | 天井较小，多为两层 |
| 文化内核 | 强调家族聚居与礼制秩序 | 商贾文化，精巧灵动 |

### 4.3 浙南（温州、丽水）山地派

| 属性 | 内容 |
|---|---|
| 核心特征 | 适应山地地形，布局灵活自由 |
| 墙体 | 卵石垒砌或夯土筑造，厚实坚固，防御性强 |
| 受闽文化影响 | 部分有土楼建筑遗存 |
| 代表 | 泰顺胡氏大院、上交垟土楼 |

### 4.4 浙东沿海派

| 属性 | 内容 |
|---|---|
| 核心特征 | 抵御台风海风，整体敦实低矮 |
| 材料 | 常用石材做墙基或墙体 |
| 屋面 | 坡度相对平缓，压顶较重 |
| 兼具 | 水乡特色与抗风防潮功能 |

### 4.5 现代浙派民居

传统与现代的融合：提炼传统特色，精简后与现代建筑语言巧妙重组。屋面平坡结合，以黑白灰色系为主，大面积玻璃门窗融入传统线条或图案元素。

## 6. 廊桥与古桥

### 6.1 泰顺廊桥

| 属性 | 内容 |
|---|---|
| 美誉 | "中国廊桥之乡" |
| 现存古廊桥 | 30余座 |

**代表廊桥**：

| 桥名 | 始建年代 | 特色 |
|---|---|---|
| 北涧桥 | 清康熙十三年（1674年） | "世界最美廊桥"，桥长51.7米，编梁式木拱廊桥 |
