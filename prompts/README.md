## 🔧 使用方法

### 查询单个镜头
```bash
cat prompts/shots/act-01/shot-001.json
```

### 列出所有镜头
```bash
ls prompts/shots/*/shot-*.json
```

### 按标签筛选
```bash
grep -l '"doro"' prompts/shots/*/shot-*.json
```

### 重新生成
```bash
python3 generate_shots.py
```

## 📋 镜头字段说明

| 字段 | 说明 |
|---|---|
| `shot_id` | 3 位数字镜头编号 |
| `act` | 所属幕 (1-6) |
| `duration_sec` | 时长（秒） |
| `image_prompt` | 图像生成提示词 |
| `video_prompt` | 视频生成提示词 |
| `audio` | 音频设计（bgm/sfx/对白/旁白） |
| `subtitle` | 字幕文本 |
| `tags` | 标签 |
| `status` | 状态：draft / review / final / archived |
