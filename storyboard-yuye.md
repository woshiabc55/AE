# 《雨夜》— 影视 PV 分镜脚本（第三部）

> **画幅**：16:9 IMAX 满画幅
> **时长**：≈ 90s（六幕 × 15s）
> **渲染基线**：3 渲 2 + IMAX 渲染 + 雨幕粒子
> **色调基线**：品红 #E91E63 + 青蓝 #00BCD4 + 暖黄窗灯 + 黑
> **视觉母题**：垂直城市 / 雨幕 / 反射 / 蒸汽
> **核心冲突**：一个雨夜快递员 vs 一个七天前已经消失的地址

## 全局技术约束

- **画面比例**：严格 16:9 IMAX 满画幅
- **运镜原则**：以横移 + 推近为主，禁用 360° 环绕（那是《对决》）；以 POV 推进叙事
- **反射与水痕**：所有玻璃面必须有雨痕或蒸汽；所有地面必须有反射
- **霓虹色规则**：品红 + 青蓝 + 暖黄三色限制；其他颜色饱和度降低 50%
- **声音层次**：雨声是底噪；粤语对白仅 1 句（茶餐厅老板）；其余用环境音推动
- **道具母题**：磁带录音机（与《归》呼应）；"谢谢"手写字（与包裹形成首尾闭环）
