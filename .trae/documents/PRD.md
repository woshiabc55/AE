# 角色设计展示页 — 产品需求文档 (PRD)

## 1. 产品概述
一个静态的、面向 PC 端的高端角色设计概念展示页,以「半面色反差 + UE5 拟真质感 + 硬线边描 + 自然光摄影」为核心视觉语言,呈现一位二次创作角色的正面大头像、半身、全身三视图。目标受众为原画/设定集爱好者、视觉设计师、3D 角色设计同行。
- 核心目的:在单个屏幕内,以高密度视觉信息展示一个角色设计的完整侧写(面部 / 服装 / 姿态 / 材质)。
- 价值:把零散的设计语言汇集成一张可读、可截屏、可二次传播的「设计板」。

## 2. 核心功能

### 2.1 用户角色
无登录、无角色区分,所有访客看到同一份展示。

### 2.2 功能模块
1. **主视觉区 (HeroCanvas)**:大尺寸画布,占据首屏 70% 高度,展示角色正面大头像。
2. **三视图分屏 (TriptychStage)**:画布上方为正面 / 侧面 / 背面三联视图,展现姿态一致性。
3. **细节材质卡 (MaterialDeck)**:3–4 张小卡,放大展示发丝、皮肤、服装、粗糙度等局部。
4. **二创标识 (DerivativeMark)**:画面右下角静态水印,声明「二创」属性。

### 2.3 页面详情
| 页面 / 区块 | 模块 | 功能描述 |
|-------------|------|----------|
| 主视觉区 | 角色大头像 | 半面色反差(左色 / 右灰度),硬线边描,自然光侧逆,UE5 拟真颗粒 |
| 主视觉区 | 环境层 | 灰白渐变背景 + 噪点纹理 + 三道高斯光斑,无边框无文字 |
| 三视图分屏 | 上半身 / 全身 | 三联并置,非镜像;姿态自然站立,上下身可拆解阅读 |
| 细节材质卡 | 局部放大 | 4 张子卡,显示粗糙度、毛发细分、布料纹路 |
| 二创标识 | 角标 | 静态文字「DERIVATIVE WORK · 二创展示」,无干扰位置 |

## 3. 核心流程
访客直接打开页面 → 首屏直接看到主视觉角色大头像(大头像 + 半面色)→ 视线自然下移到三视图分屏 → 再下移到材质细节卡 → 始终在右下角看到二创水印。无需交互、无需滚动触发(支持滚动但不依赖)。
```
[ 打开页面 ] ──▶ [ 首屏大头像 + 半面色 ] ──▶ [ 滚动 / 直接下移 ] ──▶ [ 三视图分屏 ] ──▶ [ 材质细节卡 ]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色**:深炭黑 `#0E0E10`、骨白 `#EAE6DD`、氧化红 `#B03A2E`(单点强调)
- **辅助色**:冷灰 `#6B6F76`、暖灰 `#9C8E7A`(材质层)
- **字体**:展示字 `Fraunces` / `Cormorant Garamond`(锐利衬线),正文 `Space Mono` / `JetBrains Mono`(等宽技术感)
- **按钮 / 交互**:无传统按钮,只有角标、版本号、参数标签
- **布局**:不规则网格,大块负空间 + 局部密度;三联视图采用 12 列网格的 4-4-4 切分
- **图标 / Emoji**:不引入 emoji,所有标识采用单色 SVG / 文字

### 4.2 页面设计概述
| 区块 | UI 元素 | 风格、布局、配色、字体、动效 |
|------|--------|------------------------------|
| HeroCanvas | 大头像 | 居中,占首屏 70% 高度;半面色反差(`mix-blend-mode: difference`);硬线 1px 描边;入场 fade + scale 0.96→1 |
| HeroCanvas | 背景 | 渐变 + 噪点 + 三道高斯光斑(暖 / 冷 / 中性) |
| TriptychStage | 三联 | 4-4-4 网格,角色分别在左中右;非镜像;上下身用 1px 分割线 |
| MaterialDeck | 材质卡 | 2×2 网格,4 张卡,卡内为局部放大 + 参数标签(`Roughness 0.42` 等) |
| DerivativeMark | 角标 | 右下角,等宽字体,小号大写 |

### 4.3 响应式
桌面优先(≥1280px),≥1024px 时 4-4-4 网格保持;平板(768–1024) 退化为 2 列并垂直堆叠;移动端(≤767) 退化为单列纵向时间线。

### 4.4 3D / 视觉风格引导
- **环境 / HDRI 情绪**:工业棚拍 + 阴天散射光,色温 5400K
- **光照**:主光 右上 45° 强光,补光 左下 30% 强度,轮廓光 背面 90° 冷色
- **相机**:85mm 人像镜头等效,光圈 f/2.0,焦平面在眉心
- **构图**:正面略偏 ±3°(`yaw`),体现「好奇 / 自然」姿态
- **交互 / 动效**:入场顺序 fade-in(无弹跳);鼠标在 HeroCanvas 内移动时,光斑产生 ±8px 视差
- **后处理**:颗粒 `0.06`、色差 `0.4px`、暗角 `0.25`、高光柔化
- **资源**:角色为程序化生成的 SVG / CSS 艺术(`face-construction.svg`),不使用第三方版权素材
- **性能预算**:首屏 LCP < 2.5s,CLS < 0.05,JS 体积 < 200KB(gzip)
