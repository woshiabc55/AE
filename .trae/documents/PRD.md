# 《天穹遗子》第一幕分镜展示页 - 产品需求文档

## 1. 产品概述
《天穹遗子》第一幕「展开」分镜展示页：基于 4 分钟 / 25 镜完整分镜表，以"奇幻动漫 × 神圣几何 × 史诗渺小感"为核心美学，将 25 镜的时间码、景别、运镜、画面、对白、音效与 AI 生成提示词呈现为可滚动浏览的沉浸式单页。
- 目标用户：动画导演、分镜师、AI 视频生成创作者、文学策划
- 价值：把密集的表格信息转化为具有仪式感的"档案 / 圣典"式阅读体验，同时为后续第二幕、第三幕预留扩展接口

## 2. 核心功能

### 2.1 信息角色
| 角色 | 入口方式 | 核心权限 |
|------|----------|----------|
| 访客 | 直接访问 | 浏览分镜、跳转镜头、查看技术备注 |
| 制作者 | 文档维护 | 后续追加第二幕 / 第三幕分镜（不在本版本范围） |

### 2.2 功能模块
1. **首页（单页）**：英雄区、25 镜滚动区、技术备注区、结尾字幕
2. **镜头导航**：左侧固定时间线 + 顶部镜号跳转
3. **镜头卡**：每镜一张卡，含元数据（时间码/景别/运镜/对白/音效/AI 提示词）
4. **技术档案**：单独区块呈现"手持规范/时间线错误/后侧显示"等元规则

### 2.3 页面细节
| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 首页 | 英雄区 | 巨型标题 + 神圣几何动画（斐波那契螺旋、黎曼曲面剪影） + 段落信息 |
| 首页 | 时间线导航 | 左侧固定时间线，对应 25 个镜号，点击跳转 |
| 首页 | 镜头卡 | 编号、时长、景别、运镜、画面内容、对白、音效、AI 提示词（可一键复制） |
| 首页 | 技术档案 | 表格 + 关键禁忌说明 |
| 首页 | 结尾字幕 | "走出不是流放。展开才是神判。" 仪式化呈现 |

## 3. 核心流程
1. 访客进入页面，看到英雄区与缓缓浮现的算穹元素
2. 滚动浏览 25 镜，每镜进入视口时有"沉降"式入场动画
3. 左侧时间线高亮当前镜号，可点击跳转到任意镜
4. AI 提示词卡支持"复制提示词"按钮，一键复制英文 prompt
5. 末尾技术档案 + 结尾字幕收束

## 4. 用户界面设计

### 4.1 设计风格
- **主色**：紫金（深紫 #1a0b2e → 暗金 #c9a35c → 星尘白 #e8e2d0）
- **辅色**：算穹深蓝 #0d1b3d、黑曜石 #0a0a0f、警告红 #b8324a（用于"删除线编号"）
- **字体**：
  - 标题：Cinzel / Noto Serif SC（衬线、神谕感）
  - 正文：EB Garamond / Noto Serif SC
  - 数字 / 时间码：JetBrains Mono / Space Mono（等宽、刻石感）
- **按钮样式**：半透明描边 + 内部辉光；hover 时描边转金色
- **布局**：单列长滚动（每镜一张卡），左侧时间线固定
- **图标**：使用 Lucide（如 Clock、Camera、Volume2、Wand2）以体现"圣典"质感

### 4.2 页面设计概览
| 页面 | 模块 | UI 元素 |
|------|------|----------|
| 首页 | 英雄区 | 顶部 "ACT I · 展开" 副标、巨型主标《天穹遗子》、背景斐波那契螺旋 + 黎曼曲面剪影、4:00 / 25镜 元信息、向下滚动提示 |
| 首页 | 镜头卡 | 左：镜号（巨型金色衬线数字）+ 时间码 + 时长条；右：景别 / 运镜 / 画面 / 对白 / 音效 / AI 提示词；底部"复制提示词"按钮；卡片背景为半透明玻璃 + 内部微粒 |
| 首页 | 时间线 | 25 个刻度，当前镜号放大 + 渐变描边；点击跳转 |
| 首页 | 技术档案 | 表格 + 关键禁忌"红色描边卡片" |
| 首页 | 结尾 | 黑屏 → 白色衬线字"走出不是流放。展开才是神判。" → 版权与续作预告 |

### 4.3 响应式
桌面优先（1200px+）。平板（768-1199）双列。手机（<768）单列，时间线变为顶部下拉锚点。

### 4.4 3D 场景引导（不适用）
本版本为 2D 单页 + CSS/SVG 动画，不涉及 3D 引擎。
