## 1. 产品概述

**电竞数据可视化看板** — 一个聚焦于"王者荣耀"角色/皮肤元数据与"流行游戏"全谱(手游+端游)的多维数据可视化仪表盘,为玩家、行业观察者、数据爱好者提供一站式的电竞数据洞察。

- **核心价值**:用 ECharts 将零散的公开游戏数据(角色定位、皮肤数量、游戏分类、平台分布、地区热度等)转化为直观可交互的图表,帮助用户快速理解中国及全球游戏市场结构。
- **目标用户**:游戏玩家、游戏行业从业者、数据可视化爱好者。
- **数据范围**:基于公开可获取的游戏元数据(角色名、定位、皮肤数、上线时间等)与游戏分类数据;不下载或使用受版权保护的游戏美术资源,以文字/几何符号/色块代替视觉元素,规避侵权风险。

## 2. 核心功能

### 2.1 用户角色
本项目为单角色无登录的浏览型工具,无需注册/登录体系。

### 2.2 功能模块

1. **首页 / 数据总览(Hero Dashboard)**
   - 关键指标卡片(KPI):王者荣耀英雄总数、皮肤总数、覆盖游戏数、平均评分等
   - 头条数据可视化:大尺寸主图表
2. **王者荣耀英雄分析页**
   - 英雄定位(坦克/战士/刺客/法师/射手/辅助)分布玫瑰图
   - 英雄热度/ban-pick 模拟柱状图(基于公开汇总数据)
   - 英雄性别、阵营、登场年份时间线
3. **王者荣耀皮肤分析页**
   - 皮肤品质(勇者/史诗/传说/荣耀)饼图
   - 皮肤系列归属(系列皮肤数量 Top10)水平条形图
   - 年度皮肤上新趋势折线图
4. **流行游戏总览页**
   - 平台分布(手游/端游/主机)环形图
   - 游戏类型分布(MOBA/FPS/RPG/SLG/休闲等)旭日图/treemap
   - 国内外热度对比雷达图
   - 综合评分散点图(评分 vs 在线人数)
5. **流行游戏分类页**
   - 按类别筛选(全部/手游/端游)网格 + 列表
   - 每个游戏卡片显示:名称、类型、平台、地区、评分(纯文字+几何图标)

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 总览 | KPI 卡片 | 4 个核心数字卡片(英雄数、皮肤数、游戏数、平均分) |

| 英雄分析 | 定位玫瑰图 | 展示 6 大定位英雄数量分布 |

| 皮肤分析 | 系列 Top10 | 系列皮肤数量排名 |
| 皮肤分析 | 年度趋势 | 多年皮肤上新数量折线 |

## 3. 核心流程

```
用户进入首页 → 浏览总览 KPI 与主图
   ↓
点击左侧导航(或顶部 Tab)切换
   ├─ 英雄分析 → 玫瑰图 + 热度图(支持 hover 联动)
   ├─ 皮肤分析 → 饼图 + Top10 + 趋势图
   ├─ 游戏总览 → 平台/类型/评分 三视图联动
   └─ 分类列表 → 筛选 → 卡片列表(可点击查看明细)
```

## 4. 用户界面设计

### 4.1 设计风格
- **整体气质**:深色科技感(HUD / Cyberpunk),电竞向,信息密度高但不杂乱
- **主色板**:
  - `--bg-primary`: `#0A0E1A`(深夜蓝黑)
  - `--bg-secondary`: `#111827`(卡片背景)
  - `--accent-primary`: `#00E5FF`(霓虹青,主强调)
  - `--accent-secondary`: `#FF2E63`(霓虹红,辅强调)
  - `--accent-tertiary`: `#FFD460`(金黄,品质/稀有度)
  - `--text-primary`: `#E6F1FF`
  - `--text-secondary`: `#8892B0`
- **字体**:
  - 标题/数字:`Orbitron` 或 `Rajdhani`(科技感等宽显示字体)
  - 正文:`Noto Sans SC` + `Inter`(中文/英文混排)
- **按钮/卡片**:直角或微圆角(2-4px),1px 霓虹边框 + 内发光,无拟物阴影
- **布局**:左侧固定导航栏 + 右侧主内容区,顶部细粒度面包屑;大量使用网格与负空间营造"数据驾驶舱"感
- **图标**:使用纯几何 SVG(柱/圆/方块),不使用游戏素材 logo

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 总览 | KPI 卡片 | 大号 Orbitron 数字、霓虹下划线、悬浮微动效 |
| 总览 | 主图 | 全宽 ECharts,带角标刻度与网格线 |
| 英雄分析 | 玫瑰图 | 6 瓣霓虹填充,hover 高亮某瓣放大 |
| 皮肤分析 | 饼图 | 环形 + 中心总数字 + 引导线 |
| 游戏总览 | 树图 | 矩形面积编码数量,色彩按平台渐变 |
| 分类列表 | 卡片 | 暗色卡 + 顶部细色条(平台标识) + 元数据网格 |

### 4.3 响应式
- **Desktop-first**:1440px 设计基准
- 平板(≥768px):导航折叠为顶部 Tab,卡片网格 2 列
- 移动端(≥375px):单列堆叠,图表保持可横向滚动或简化

### 4.4 视觉细节
- 背景:深色 + 极细网格底纹(20px × 20px,透明度 0.04) + 顶部 radial-gradient 蓝色光晕
- 微动效:首屏 KPI 数字计数动画、图表入场 stagger、卡片 hover 时边框霓虹流光
- 装饰:角落 L 形装饰线、刻度感分隔条
