## 1. 产品概述

「像素骑士：暗影征伐」是一款采用「像素 2 渲 3」工艺的网页端横板动作砍杀独立游戏。在复古像素美术之上叠加多层视差、动态光照、体积雾与 CRT 后处理，营造类 HD-2D 的立体纵深感，玩家操控骑士在月夜古堡中挥剑斩杀亡灵军团。

- 核心目标：交付一个即开即玩、单局 3-5 分钟的高完成度横板砍杀 demo，展现 2D 像素 + 3D 光影的独立游戏美学
- 目标用户：独立游戏玩家、像素动作游戏爱好者、横板 ACT 受众
- 市场价值：验证「像素 2 渲 3」在 Web 平台的可行性，作为可扩展的独立游戏项目骨架

## 2. 核心功能

### 2.1 玩家角色
| 角色类型 | 战斗风格 | 核心能力 |
|---------|---------|---------|
| 骑士 | 近战剑术 | 移动、跳跃、三段连斩、冲刺闪避、格挡反击 |

### 2.2 功能模块
1. **战斗系统**：三段连斩、击退、连击计数、暴击、冲刺无敌帧
2. **敌人系统**：骷髅战士 AI（巡逻 / 追击 / 攻击 / 受击 / 死亡）
3. **关卡场景**：多层视差古堡背景、动态光照、可破坏物件
4. **HUD 系统**：血量、连击数、分数、波次提示
5. **后处理特效**：体积光、CRT 扫描线、暗角、色调、粒子（血、火花、尘）

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 标题界面 | 启动画面 | 游戏标题、开始按钮、操作说明、像素动态背景 |
| 游戏界面 | 战斗画布 | Canvas 实时渲染玩家、敌人、场景、光照、粒子 |
| 游戏界面 | HUD 层 | 血条、连击计数、分数、波次倒计时 |
| 结算界面 | 胜负画面 | 胜利 / 死亡结算、分数统计、重新开始 |

## 3. 核心流程

**主游戏循环**：玩家进入标题界面 → 点击开始 → 进入古堡关卡 → 波次刷新骷髅敌人 → 挥剑连斩清场 → 累计连击与分数 → 清空波次胜利 / 血量归零失败 → 结算 → 重新开始。

```mermaid
flowchart TD
    "A[标题界面]" --> "B[点击开始]"
    "B" --> "C[进入古堡关卡]"
    "C" --> "D[波次刷新敌人]"
    "D" --> "E[移动 / 跳跃 / 冲刺]"
    "E" --> "F[三段连斩攻击]"
    "F" --> "G[命中敌人 + 连击计数]"
    "G" --> "H{波次清空?}"
    "H -- 否 --> "D"
    "H -- 是 --> "I[胜利结算]"
    "C --> J{血量归零?}"
    "J -- 是 --> "K[死亡结算]"
    "I --> "L[重新开始]"
    "K --> "L"
    "L --> "B"
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深夜墨蓝 `#0a0e1a` 底 + 月光银青 `#8ab4c4` + 暖焰橙红 `#ff5733`（剑光 / 血）+ 亡灵幽绿 `#3ddc84`
- **次色调**：金黄 `#ffd23f`（连击高亮）、紫色 `#7b5ea7`（魔法 / 体积光）
- **按钮风格**：像素厚边框按钮（4px 描边 + 内阴影），按下位移模拟 3D 凸起
- **字体**：标题用 `Press Start 2P`（像素显示字体），HUD 用 `VT323`（复古终端字体）
- **布局风格**：全屏 Canvas 战斗区，HUD 浮层叠加四角，标题界面居中构图
- **图标风格**：像素剑 / 骷髅 / 心形 SVG 图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 标题界面 | 标题画面 | 像素标题大字、骑士剪影、火把光晕、开始按钮、操作提示 |
| 游戏界面 | 战斗画布 | 多层视差古堡、骑士精灵、骷髅敌人、动态光照、粒子特效 |
| 游戏界面 | HUD 左上 | 血量条（心形像素）、连击数、分数 |
| 游戏界面 | HUD 顶部 | 波次进度、敌人数 |
| 结算界面 | 结算画面 | 胜利 / 失败标题、最终分数、最高连击、重玩按钮 |

### 4.3 响应式
- 桌面优先（≥1024px）：1280×720 固定逻辑分辨率，自适应缩放铺满
- 平板（768-1023px）：保持 16:9 比例缩放
- 移动端（<768px）：虚拟摇杆 + 攻击按钮触控适配

### 4.4 2.5D / 3D 场景指引
- **环境氛围**：月夜古堡废墟，冷月光主光 + 火把暖光点光源，地面薄雾
- **光照设置**：全局方向光（月光，冷蓝）+ 动态点光源（火把、剑光、命中爆光）+ 角色脚下椭圆阴影
- **摄像机**：固定横板视角，多层视差（远景天空 → 中景城堡剪影 → 近景柱列 → 前景雾），玩家移动时各层以不同速率滚动制造纵深
- **构图**：骑士始终位于画面左 1/3，敌人从右侧涌入，留出战斗空间
- **后处理**：CRT 扫描线 + 轻微曲率、暗角 vignette、暖色色调映射、命中时屏幕震动 + 红色闪屏
- **资产来源**：全部由代码程序化生成像素精灵（Canvas 绘制），无外部图片依赖
- **性能预算**：60 FPS，同屏 ≤ 8 敌人，粒子 ≤ 200

## 5. 操作说明
- **A / D 或 ← / →**：左右移动
- **W / ↑ / 空格**：跳跃（支持二段跳）
- **J / 鼠标左键**：攻击（三段连斩）
- **K / Shift**：冲刺闪避（带无敌帧）
- **L / 鼠标右键**：格挡（短窗口反击）
- **Q/E/R/F/V/C/X/Z**：8 个主动技能（消耗专注值）

## 6. 剧情设定

永夜降临。死灵领主 **永夜之主 Morgrim** 以「蚀日之冠」熄灭了太阳，世界陷入不死的暗影。

你是最后一位圣骑士，手持祖传圣剑 **晨曦之刃**，背负唤醒黎明的使命。穿越四境——亡者墓地、腐沼深渊、寒霜要塞、永夜王座，斩断四道死灵封印，最终在王座之巅与 Morgrim 决战。

每击败一位 Boss，晨曦之刃吸收其封印之力，解锁新技能。

## 7. 技能系统

### 7.1 资源：专注值（Focus）
- 最大值 100，缓慢自动回复（8/s），命中敌人额外回复（2/次）
- 释放技能消耗专注值，不足则无法释放

### 7.2 主动技能（8 个）
| 键位 | 技能名 | 消耗 | 效果 |
|------|--------|------|------|
| Q | 旋风斩 | 12 | 360° 范围斩击，击退周围敌人 |
| E | 盾击 | 15 | 盾牌重击，眩晕敌人 1.2s |
| R | 突进斩 | 18 | 朝向突进斩击，路径敌人受伤害 |
| F | 圣光弹 | 10 | 发射远程光弹，穿透敌人 |
| V | 陨星斩 | 25 | 跃起下劈，落地 AOE + 震屏 |
| C | 嗜血狂怒 | 30 | 8s 内攻速 +40%、吸血 25% |
| X | 雷霆审判 | 28 | 召唤 3 道落雷连锁打击 |
| Z | 晨曦终焉 | 60 | 终极技：全屏光柱，巨量神圣伤害 |

### 7.3 被动技能（6 个，击败 Boss 解锁）
| 名称 | 解锁章节 | 效果 |
|------|---------|------|
| 吸血 | 第一章 | 攻击回复 5% 伤害的血量 |
| 暴击大师 | 第一章 | 暴击率 +25%，暴击 1.8x |
| 迅捷之风 | 第二章 | 移速 +15%，冲刺冷却 -30% |
| 钢铁意志 | 第二章 | 受到伤害减免 20% |
| 连击狂热 | 第三章 | 连击倍率上限 +50% |
| 光之护佑 | 第三章 | 致死时复活一次（每场战斗 1 次），恢复 40% HP |

## 8. 关卡与 Boss

### 8.1 第一章 · 亡者墓地
- **场景**：月夜古堡墓地（当前实现基础）
- **杂兵**：骷髅斥候、骸骨弓手（远程）
- **Boss**：**骸骨将军 Kael** —— 复活的禁卫军统领
  - 阶段一：三连斩 + 冲撞
  - 阶段二（50%）：召唤 2 个骷髅兵 + 旋风斩

### 8.2 第二章 · 腐沼深渊
- **场景**：毒雾沼泽，绿色氛围，浮岛木桥
- **杂兵**：沼泽史莱姆（分裂）、毒蝇（飞行）
- **Boss**：**沼泽巨魔 Bilefang** —— 巨型腐尸
  - 阶段一：重拳砸地（AOE）+ 投掷毒球
  - 阶段二：狂暴，跳砸 + 毒雾扩散

### 8.3 第三章 · 寒霜要塞
- **场景**：风雪冰原要塞，蓝色冷调
- **杂兵**：冰霜亡灵、冰元素（投射冰锥）
- **Boss**：**霜誓骑士 Veyra** —— 叛变的圣骑士
  - 阶段一：镜像骑士对战，剑技对拼
  - 阶段二：冰封领域，地面结冰打滑 + 冰柱降临

### 8.4 第四章 · 永夜王座
- **场景**：漆黑王座大殿，紫色魔法光
- **杂兵**：暗影骑士、怨灵（无敌帧穿透）
- **Boss**：**永夜之主 Morgrim**（双阶段）
  - 阶段一：暗影法师，传送 + 暗影弹幕 + 召唤
  - 阶段二（50%）：化身暗影巨魔，全屏 AoE + 蚀日激光

## 10. 解锁与成长
- 每章击败 Boss 后解锁 2 个被动技能
- 章节间自动满血并恢复专注值
- 死亡从当前章节开头重开（不回标题）
