# 《哨兵之桥》电影剧本展示页 — PRD

## 1. 产品概述

将《哨兵之桥》末日废土题材12章节电影剧本以电影化、沉浸式单页HTML的方式呈现，让读者像翻阅一本"会呼吸的剧本"一样阅读作品。
- 目标用户：剧本读者、影视爱好者、末日题材粉丝
- 核心价值：通过电影化的视觉语言（镜头语言、色彩氛围、动效节奏）让文字剧本具备"分镜感"与"可观赏性"

## 2. 核心功能

### 2.1 内容结构
| 区域 | 内容 | 功能 |
|------|------|------|
| 序幕（裂缝之上） | 序章 | 世界观与主人公引出 |
| 主体（10个章节） | 第一章至第十章 | 故事核心推进 |
| 终章（桥梁） | 第十一章 + 终章 | 收束与传承 |
| 附录区 | 镜头语言参考、主题关键词、核心对白索引 | 创作笔记 |

### 2.2 页面模块
1. **开场封面 (Hero/Cover)**：黑屏渐显标题"哨兵之桥"，副标题"末日废土 · 12章节电影剧本"，底部署名，整体压暗的氛围。
2. **章节目录 (TOC)**：以横向滚动/竖向列表形式列出全部章节，支持点击跳转，含氛围色块标识。
3. **章节阅读区 (Chapter Sections)**：每个章节独立section，顶部"第X章" + 章名 + 章节导语，下方为多个场景卡片。
4. **场景卡片 (Scene Card)**：以"分镜脚本"的视觉语言呈现：场景号、内外景、景别/调度提示、动作描写、氛围标签、对白。
5. **附录交互表 (Appendix Tables)**：镜头语言参考、主题关键词弧线、核心对白索引，以视觉化表格呈现。
6. **结尾黑屏字幕 (End Credits)**：淡出 + 字幕升起"在废土上，最坚固的桥梁不是混凝土，是人。"

### 2.3 交互能力
- 章节目录点击 → 平滑滚动到对应章节
- 顶部固定进度条（阅读进度）
- 阅读过程中的环境氛围：背景随章节切换色调（序章灰黄→夜战深黑→终章暖橙）
- 场景卡片的微动画（淡入/上浮/字幕式逐行显示）
- 鼠标移动时的尘埃/灰烬粒子效果
- 顶部"返回目录"快捷按钮

## 3. 核心流程

```mermaid
flowchart TD
    A[进入页面] --> B[封面渐显 + 标题]
    B --> C[点击"开始阅读"]
    C --> D[滚动浏览章节目录]
    D --> E[逐章阅读场景卡片]
    E --> F{是否需要回顾镜头语言?}
    F -- 是 --> G[滚到附录区查阅]
    F -- 否 --> H[继续阅读下一章]
    G --> H
    H --> I[终章结束]
    I --> J[字幕升起 + 全剧终]
```

## 4. 用户界面设计

### 4.1 设计风格

- **整体调性**：电影化、孤独、废土、史诗感。介于"末日摄影集"与"电影分镜书"之间。
- **主色板**：
  - 废土黄 `#a89a7b` / 灰土 `#3a342c` / 焦土深褐 `#1a1612`
  - 地底岩浆红 `#7a1a1a` / 暗火橙 `#c2410c`
  - 辐射冷灰 `#8a8a85` / 骨白 `#d4cfc2`
  - 终章暖光 `#e8a04a` / 新生绿 `#5a7340`
- **辅助色**：警示红 `#dc2626`、守护铜 `#b08864`
- **字体**：
  - 标题字体：一种具有工业感/电影感的衬线或展示字体（推荐 `Cormorant Garamond` 或 `Noto Serif SC`）
  - 正文字体：`Noto Sans SC` / `Inter` 备选
  - 强调/代码：等宽字体 `JetBrains Mono`
- **按钮风格**：极简描边/方形切割感，避免圆角，呼应末日的硬朗
- **布局**：左侧固定"章节锚点"，右侧阅读内容流；移动端折叠为顶部抽屉
- **氛围元素**：
  - 背景噪点纹理（纸/沙）
  - 偶发尘埃粒子漂浮
  - 章节切换时的色温过渡
  - 文字在滚动时的轻微视差

### 4.2 页面设计概览

| 区域 | 模块 | UI 元素 |
|------|------|----------|
| 封面 | 标题/副标题/开始按钮 | 黑屏 + 渐显 + 缓慢放大标题、底部细线 |
| 目录 | 章节列表 | 数字 + 章名 + 短导语 + 色块条 |
| 章节头 | 章号 + 章名 | 大字号、左对齐、一根细横线、英文小标 |
| 场景卡 | 场号+内外景+景别+正文+氛围标签 | 顶/左侧小标识 + 段落块 + 底部氛围字 |
| 对白 | 引号 + 角色名 | 角色名小字、引文加粗或斜体 |
| 附录 | 三张表格 | 表格行交替底色 + 关键词高亮 |
| 字幕 | 全剧终引言 | 大字居中、缓慢淡入、滚动上升 |

### 4.3 响应式适配
- 桌面优先（1280px+ 最佳）
- 平板（768-1280px）调整网格与字号
- 移动端（<768px）章节锚点折叠为顶部抽屉，单列流式阅读

### 4.4 氛围场景引导

- **序章/战前**：偏冷灰 + 黄昏暗橙；低音量的"风沙"感背景（可选纯CSS动画）
- **第一章夜战**：压暗、煤油灯橙黄摇曳的CSS box-shadow动画
- **闪回实验室**：短暂切到高亮、白色偏冷
- **第六-七章夜袭**：深黑 + 绿色畸变体光点
- **终章新生**：暖橙夕阳 + 一抹绿意

## 5. 验收标准
- 12个章节（序章+1-11+终章）全部正确呈现
- 章节目录、场景卡片、附录表格内容完整
- 滚动动效、章节锚点跳转、进度条可用
- 桌面/平板/移动端均能正常阅读
- 整体氛围符合"末日废土+电影化"调性
