# PRD 产品需求文档 — 大荧幕·觉醒 (IMAX 大银幕PV)

## 1. 项目概述

### 1.1 项目名称
**《觉醒 · The Awakening》— IMAX 大银幕PV 整合分镜框架**

### 1.2 项目定位
一款基于 React 的"影视级PV分镜生产与可视化框架"。以**电影级IMAX巨幕开场**为视觉目标，将用户提供的散乱文段整合为可交互、可预览、可分镜调试的6幕18镜完整分镜脚本及对应的实时可视化视频播放体验。

### 1.3 目标用户
- **电影/动画导演**：直观预览分镜节奏、镜头语言、运镜手法
- **视效预览师 (Previs Artist)**：快速验证镜头、特效、转场
- **制片/投资人**：以"可看可听"的方式评审脚本
- **剧本统筹**：整合散乱文本，结构化输出可执行分镜表

### 1.4 核心价值
将文字描述的"宏观画面 / 圣像 / 农夫觉醒 / 行星坠落 / 太空巨兽 / 水墨金龙"等复杂镜头语言，通过 React 框架**渲染为可交互的视频级预览**，并提供分镜卡片表、运镜可视化、音效时间轴三件套。

## 3. 功能需求

### 3.1 核心功能
| ID | 模块 | 描述 | 优先级 |
|----|------|------|--------|
| F-01 | IMAX巨幕播放器 | 16:9 IMAX画幅，支持播放/暂停/进度条/时间码 | P0 |
| F-02 | 分镜卡片表 | 6幕18镜，可滚动查看，镜号高亮联动 | P0 |
| F-03 | 运镜可视化 | 根据"地面仰拍/急速拉升/环绕长镜头"等运镜实时变换视角 | P0 |
| F-04 | 粒子/水墨/IMAX渲染 | Canvas/CSS混合：粒子、空间褶皱、水墨线条、金龙 | P0 |
| F-05 | 音效时间轴 | 18个音效节点，与镜头时间码同步触发 | P0 |
| F-06 | 黑屏与色调系统 | 严格低饱和《沙丘》色调，最终幕黑屏收束 | P1 |
| F-07 | 关键运镜动画 | 急速拉升/360度环绕/急速后退/慢动作 | P1 |
| F-08 | 角色状态切换 | 农夫↔觉醒↔金龙 三态形变 | P1 |

### 3.2 视觉规范
- **画幅**：16:9 IMAX，黑色遮幅，上下细黑边
- **色调**：严格《沙丘》低饱和，黄褐/灰青/赭石，少量金色高光
- **字体**：标题—汉字宋体派生（如"思源宋体"/"Noto Serif SC"）；数字—等宽未来感（如"JetBrains Mono"）
- **运动**：缓入缓出 0.6s cubic-bezier(0.65, 0, 0.35, 1)
- **光感**：强逆光剪影、空气透视体积光

### 3.3 交互规范
- 自动播放按钮 / 进度条拖拽 / 镜号点击跳转
- 鼠标悬停分镜卡片：显示该镜头的视觉预览
- 键盘 Space 播放/暂停，← → 切换镜号

---

## 4. 非功能需求
- 渲染：Canvas 2D 为主 + CSS 3D Transform + 少量 SVG
- 性能：60fps 目标，< 1.5s 启动
- 响应式：1280px+ 桌面优先
- 兼容：Chrome / Edge / Safari 最新版
