# 航线指令·烬海初航 - 产品需求文档

## 1. 产品概述

《航线指令·烬海初航》第一章的IMAX视觉叙事剧本互动展示页面。以"墨染星海、金缮瓷痕、液态罗盘"为核心视觉语言，将约2000字的中式科幻太空史诗以电影化分镜形式呈现。
- 主要目的：让观众/读者沉浸式阅读剧本，感受"裂痕是光进入的地方"的核心隐喻
- 目标用户：科幻文学爱好者、影视剧本策划、中式美学设计研究者

## 2. 核心功能

### 2.1 内容模块
1. **开场墨染动画**：黑场 → 墨滴坠入清水 → 晕染为星海 → 穿越金属坟场
2. **角色档案模块**：林砚秋 / 司南 / 陈牧野 三位角色独立展示，含视觉符号（碎瓷珠、罗盘投影、青瓷胸针）
3. **四幕分镜剧场**：
   - 第一幕：烬海（0:00-1:00）
   - 第二幕：古码（1:00-2:30）
   - 第三幕：抉择（2:30-3:30）
   - 第四幕：初航（3:30-4:00）
4. **IMAX视觉设计备忘录**：6组镜头×视觉核心×中式科幻元素对照表
5. **尾幕字幕**：金缮疤痕隐喻的收束句

### 2.2 交互功能
- 滚动驱动的墨染晕染与星空推进
- 章节导航（顶部时间轴跳转）
- 角色卡片悬停3D翻转与符号浮现
- 段落文本逐字显现（与时长4分钟节奏同步）
- 鼠标移动产生星尘跟随粒子效果
- 青铜罗盘液态指针在抉择幕分裂的CSS动效

## 3. 核心流程

1. 用户进入 → 黑场 + 钟声低鸣
2. 墨滴动画 → 推进至金属坟场俯冲
3. 滚动至角色档案 → 三角色依次出现
4. 滚动触发四幕分镜 → 每幕按"IMAX视觉→对白→声音"三层信息层级展开
5. 终幕字幕 → 返回地球海岸线
6. 可选：点击"再次入海"重置旅程

## 4. 用户界面设计

### 4.1 设计风格
- **主色**：深空黑 `#0a0d14`、瓷釉靛蓝 `#1a3a6c`、金缮赤金 `#c9a35c`、青铜锈绿 `#4a6b5b`、坟场灰 `#2a2e35`
- **字体**：
  - 中文标题：思源宋体 Heavy / Ma Shan Zheng（行楷，用于题字）
  - 正文：思源宋体 Regular
  - 时间码/编号：JetBrains Mono / 等宽数字
- **布局**：全屏沉浸式滚动剧场，单列纵深，章节间全屏切换
- **质感**：颗粒噪点叠加 + 金箔光斑 + 极淡的青花暗纹底纹
- **动效**：墨水晕染（filter:blur + 渐变蒙版）、碎瓷珠零重力漂浮（CSS animation）、青铜指针分裂（transform:rotateX）

### 4.2 页面设计概览

| 页面 | 模块 | UI元素 |
|------|------|--------|
| 开场幕 | 墨染启动 | 全屏黑场 + 缓慢扩散的墨滴 + 钟声触发后星海浮现 |
| 角色幕 | 三角色档案 | 错落卡片布局，悬停时符号（瓷珠/罗盘/胸针）放大 |
| 第一幕 | 烬海 | 360°星图旋转 + 朱漆编号闪烁 + 碎瓷珠零重力声波 |
| 第二幕 | 古码 | 符号雨 + 瞳孔倒影 + 闪回江南雨巷 |
| 第三幕 | 抉择 | 罗盘指针分裂双轨 + 蓝光人像分割 |
| 第四幕 | 初航 | 纳米披风展开 + 十二灯塔亮起 + 地球收束 |
| 设计备忘 | 镜头对照表 | 表格化卡片，金缮纹理分隔线 |

### 4.3 响应式
- Desktop-first，1440px 起步优化
- 平板自适应（768px）双列角色卡
- 移动端单列（375px）保留所有动效，简化粒子数

## 4.4 视觉场景指引
- **环境/氛围**：极深空黑为底，金光与靛蓝作主光源
- **光影**：定向暖光（角色）+ 环境冷光（星图）+ 闪烁（朱漆/青花）
- **构图**：中心对称（罗盘）+ 极端纵深（坟场俯冲）+ 黄金分割留白
- **后期**：颗粒噪点 + 暗角晕影 + 玻璃反光 + 油画笔触
- **性能预算**：60fps，单页面 < 2MB 资源
