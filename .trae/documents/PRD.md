# PRD - Skill Forge 技能工坊

## 1. 产品概述
Skill Forge 是一个汇集大量前端"技能工具"的创意工坊网站，每个工具都是一个小巧的HTML/CSS/JS演示，展示一种独特的界面效果或交互技巧。
- 主要用途：让设计师/前端开发者浏览、复制、收藏大量可独立运行的HTML界面技能示例
- 目标用户：前端开发者、UI设计师、创意编程爱好者
- 核心价值：在一个页面中集中展示几十种不同视觉风格的HTML技能工具，提供即时预览与代码查看

## 2. 核心功能

### 2.1 工具分类
- **视觉效果类**：玻璃拟态、噪点纹理、渐变网格、3D 卡片、滚动视差
- **交互组件类**：磁性按钮、拖拽卡片、键盘可视化、波形发生器
- **动画类**：粒子背景、字重动画、滚动触发动画、鼠标跟随
- **生成器类**：配色生成器、字体搭配生成器、CSS 噪点生成器、阴影生成器
- **趣味实验类**：故障艺术、像素化、ASCII 转换、磁场鼠标

### 2.2 功能模块
1. **首页 / 工具展示页**：网格化的卡片布局，每张卡片是一个独立工具的入口
2. **工具详情页**：展示工具效果与可复制的代码
3. **导航与筛选栏**：分类切换、关键词搜索
4. **关于页**：项目介绍与设计灵感

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|--------------------|
| 首页 | Hero 横幅 | 巨型字体标题 + 动态背景，明示"大量HTML技能工具" |
| 首页 | 分类筛选 | 顶部水平分类标签，支持单选切换 |
| 首页 | 工具网格 | 卡片化布局，每张卡片展示一个工具的实时预览与名称 |
| 首页 | 全局搜索 | 输入关键词过滤工具 |
| 工具详情 | 预览区 | 工具本身的全尺寸预览 |
| 工具详情 | 代码区 | 展示 HTML/CSS/JS 源码并支持一键复制 |
| 工具详情 | 元信息 | 显示分类、标签、创建时间 |
| 关于 | 项目说明 | 介绍工坊理念与作者 |

## 3. 核心流程
用户进入首页 → 浏览分类 → 点击感兴趣的工具卡片 → 进入详情页查看效果与源码 → 复制代码到自己的项目。
[可选流程] 用户可在首页直接使用工具的简化版本进行操作。

## 4. 用户界面设计

### 4.1 设计风格
- **主色**：深墨黑 `#0a0a0a` 背景
- **强调色**：电光黄 `#f0ff00`、霓虹粉 `#ff3da5`、青蓝 `#00e5ff`
- **辅助色**：米白 `#f5f1e8` 用于文本对比
- **字体**：显示字体用 `"Fraunces"` 或 `"Editorial New"` 风格；正文字体用 `"JetBrains Mono"` 强调代码气质
- **按钮风格**：纯黑底白字 + 黄色高亮态，2px 硬边框 + 偏移阴影
- **布局风格**：网格化卡片，模块间使用粗细分隔线，强调瑞士国际主义风格
- **图标风格**：单色线性图标与几何符号

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页 | Hero | 巨号衬线字体 + 旋转标语 + 黄色色块动画 |
| 首页 | 工具网格 | 卡片宽高比 1:1，hover 时卡片上浮 + 边框变黄 |
| 工具详情 | 预览 | 暗背景画布占据上半屏 |
| 工具详情 | 代码 | 等宽字体，行号显示，黄色选中高亮 |

### 4.3 响应式
- 桌面优先（1280px+）
- 自适应到平板（768px-1280px）：网格 3 列
- 自适应到手机（<768px）：网格 1 列，导航折叠

## 5. 非功能性要求
- 单页加载性能优秀，工具卡片的预览使用 iframe 或内联 mini 渲染
- 所有工具均为纯前端实现，无后端依赖
- 代码块语法高亮（轻量级自实现或 highlight.js）
