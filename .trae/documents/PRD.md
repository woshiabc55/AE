## 1. 产品概述

「MotionCanvas」是一款面向动画设计师、前端开发者和创意工作者的网页端 SVG 图层动画设计与 2D 数字动画创作工具。用户可通过图层化管理 SVG 部件，在可视化时间线上编排关键帧动画，支持场景切换与逻辑画布交互，实现从静态 SVG 到动态交互式动画的全流程创作。

- 核心目标：让用户在浏览器内完成「SVG 图层设计 → 部件编排 → 时间线关键帧 → 场景切换 → 2D 数字动画导出」的完整创作闭环
- 目标用户：UI/UX 动效设计师、前端动效开发者、2D 动画创作者、交互原型制作者
- 市场价值：填补浏览器端「SVG 图层动画 + 场景时间线 + 逻辑画布」一体化工具空白，降低 2D 矢量动画创作门槛

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 创作者 | 无需注册（本地数据） | SVG绘制、图层管理、动画编排、场景切换、导出 |

### 2.2 功能模块
1. **SVG 画布工作台**：矢量绘制、图层面板、属性编辑、对齐辅助
2. **图层面板**：图层增删、排序、显隐、锁定、分组、混合模式
3. **部件场景时间线**：关键帧编辑、缓动曲线、图层轨道、播放控制
4. **场景管理器**：多场景创建、切换、场景过渡效果
5. **逻辑画布**：交互事件绑定、条件触发、变量控制
6. **导出面板**：SVG 动画导出、CSS 动画代码、GIF/视频导出

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| SVG 画布工作台 | 矢量绘制区 | 支持矩形、圆形、路径、文字、线条等 SVG 基础图形绘制 |
| SVG 画布工作台 | 图层面板 | 左侧图层列表，支持拖拽排序、显隐切换、锁定、分组 |
| SVG 画布工作台 | 属性编辑器 | 右侧面板，编辑选中元素的填充、描边、透明度、变换等属性 |
| SVG 画布工作台 | 对齐辅助 | 参考线、网格、智能对齐、吸附 |

| 导出面板 | CSS 代码 | 生成 @keyframes CSS 动画代码 |

## 3. 核心流程

**主创作流程**：用户进入工作台 → 在画布上绘制 SVG 图形 → 图层面板自动创建对应图层 → 组织图层层级与分组 → 切换到时间线模式 → 为各图层添加关键帧 → 编辑缓动曲线 → 创建多场景 → 设定场景过渡 → 绑定交互逻辑 → 预览播放 → 导出动画。

```mermaid
flowchart TD
    "A[进入工作台]" --> "B[SVG 图形绘制]"
    "B" --> "C[图层管理与分组]"
    "C" --> "D[时间线关键帧编排]"
    "D" --> "E[缓动曲线编辑]"
    "E" --> "F[场景创建与过渡]"
    "F" --> "G[逻辑画布交互绑定]"
    "G" --> "H[预览播放调试]"
    "H" --> "I[导出动画文件]"
```

**场景切换流程**：用户在场景管理器中新建场景 → 复制/清空画布 → 编排新场景关键帧 → 设定两场景间过渡效果 → 播放预览过渡动画。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝灰背景 `#0f1117` + 亮青强调色 `#00e5ff` + 珊瑚橙辅助色 `#ff6b6b`，营造专业动效工具的科技感与创意活力
- **次色调**：图层色标 8 色（青、橙、紫、绿、粉、黄、蓝、红），面板背景 `#1a1d27`
- **按钮风格**：圆角矩形（6px），主按钮带渐变高亮，工具按钮半透明悬浮
- **字体**：标题用 `Space Grotesk`（几何感显示字体），正文用 `IBM Plex Sans`（清晰专业）
- **布局风格**：顶部工具栏 + 左侧图层面板 + 中央画布 + 右侧属性面板 + 底部时间线，类 Figma/AE 布局
- **图标风格**：线性风格 SVG 图标，2px 描边，与工具界面统一

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| SVG 画布工作台 | 矢量画布 | 无限画布（可缩放平移）、网格背景、辅助线、选择框 |
| SVG 画布工作台 | 顶部工具栏 | 选择/矩形/圆形/路径/文字/画笔/吸管工具，缩放控制 |
| SVG 画布工作台 | 图层面板 | 图层列表、拖拽手柄、眼睛图标（显隐）、锁图标、色标 |
| SVG 画布工作台 | 属性面板 | 填充色选择器、描边设置、透明度滑块、变换输入框 |
| 部件场景时间线 | 时间轴 | 横向时间标尺、图层轨道、菱形关键帧、播放头竖线 |
| 部件场景时间线 | 缓动编辑 | 贝塞尔曲线画布、预设缓动按钮、数值输入 |
| 场景管理器 | 场景条 | 底部缩略图横向滚动、当前场景高亮、+号添加 |
| 逻辑画布 | 节点编辑 | 事件节点、动作节点、变量节点、连线 |

### 4.3 响应式
- 桌面优先（≥1440px）：完整五区布局
- 笔记本（1024-1439px）：属性面板折叠为抽屉
- 平板及以下：暂不支持，专注桌面体验

### 4.4 3D 场景指引
本项目为 2D SVG 动画工具，不涉及 3D 场景。但支持 2.5D 透视变换效果（通过 CSS transform3d）。
