## 1. 产品概述

「植物大战僵尸游戏模板」是一款基于浏览器的塔防游戏模板，使用纯 SVG 绘制植物、僵尸、场景与 UI 设定图标，提供可运行的游戏核心循环与可扩展的素材体系。

- 核心目标：提供一个可直接运行、易于二次开发的 PvZ 风格游戏模板，展示如何用 SVG 实现完整的游戏设定
- 目标用户：前端学习者、独立游戏开发者、需要 SVG 游戏素材的设计师
- 市场价值：作为教学示例与项目起点，降低 HTML5 塔防游戏开发门槛

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 玩家 | 无需注册 | 种植植物、收集阳光、抵御僵尸、暂停/重置游戏 |

### 2.2 功能模块
1. **游戏主场景**：5×9 草地棋盘、背景、草坪格高亮
2. **植物系统**：向日葵、豌豆射手、坚果墙，每种植物拥有独立 SVG 与能力
3. **僵尸系统**：普通僵尸、路障僵尸、铁桶僵尸，带行走与受击动画
4. **子弹与阳光**：豌豆子弹飞行、阳光生成与收集
5. **设定面板**：植物卡槽、阳光计数、波次提示、开始/暂停/重置按钮
6. **SVG 图鉴**：展示所有植物、僵尸、道具的 SVG 设定

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主界面 | 棋盘 | 5 行 9 列草地，点击空格种植当前选中植物 |
| 游戏主界面 | 植物卡槽 | 显示可种植植物、阳光消耗、冷却状态 |
| 游戏主界面 | 阳光计数器 | 顶部显示当前阳光数量 |
| 游戏主界面 | 控制按钮 | 开始波次、暂停/继续、重置关卡 |
| 图鉴页面 | SVG 设定展示 | 分类展示植物、僵尸、场景、UI 的 SVG 原画 |

## 3. 核心流程

**主游戏流程**：玩家进入游戏 → 从卡槽选择植物 → 点击棋盘格子种植 → 向日葵自动生成阳光，豌豆射手攻击同排僵尸 → 僵尸从右侧向左侧行进 → 僵尸被消灭或到达最左侧则游戏结束/扣血 → 完成一波后进入下一波。

```mermaid
flowchart TD
    "A[进入游戏]" --> "B[选择植物卡槽]"
    "B" --> "C[点击棋盘种植]"
    "C" --> "D[植物生效]"
    "D" --> "E[生成阳光 / 发射豌豆]"
    "E" --> "F[僵尸从右侧出现]"
    "F" --> "G[豌豆命中僵尸]"
    "G" --> "H{僵尸生命值 > 0?}"
    "H" -->|是| "F"
    "H" -->|否| "I[僵尸死亡]"
    "I" --> "J{波次结束?}"
    "J" -->|否| "F"
    "J" -->|是| "K[进入下一波]"
```

**图鉴流程**：玩家点击图鉴入口 → 按植物/僵尸/场景/UI 分类浏览 → 查看每个 SVG 设定的详细参数与用途。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：草坪绿 `#6ead3b` + 天空蓝 `#87ceeb` + 阳光金 `#ffd700`
- **次色调**：植物绿 `#4caf50`、僵尸灰 `#8f9a7e`、血液红 `#e53935`
- **按钮风格**：圆角矩形，主按钮带轻微立体阴影，卡槽按钮带选中边框
- **字体**：标题使用 `Creepster`（僵尸恐怖趣味感），正文使用 `Nunito`（圆润易读）
- **布局风格**：顶部 HUD + 左侧卡槽 + 中央棋盘，图鉴为卡片网格
- **图标风格**：手绘感扁平 SVG，植物圆润、僵尸略带歪斜呆萌感

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 游戏主界面 | 棋盘 | 草地背景、网格线、悬停高亮、种植后的植物 |
| 游戏主界面 | HUD | 阳光图标与数量、波次进度、生命值 |
| 游戏主界面 | 卡槽 | 植物缩略图 SVG、阳光价格、冷却遮罩 |
| 图鉴页面 | 分类标签 | 植物 / 僵尸 / 场景 / UI |
| 图鉴页面 | 设定卡片 | 大图标 SVG、名称、描述、参数 |

### 4.3 响应式
- 桌面优先（≥1024px）：完整布局，棋盘固定比例
- 平板（768-1023px）：卡槽改为横向滚动
- 移动端（<768px）：棋盘缩放，卡槽与 HUD 紧凑排列

### 4.4 3D 场景指引
本项目为 2D SVG 游戏，不涉及 3D 场景。
