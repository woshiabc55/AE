## 1. 产品概述

「高数卷宗」是一个面向高校学生与备考者的《高等数学》试卷与复习提纲展示平台，以学术期刊式的编辑设计呈现真题试卷、章节提纲与考点脉络，让数学复习兼具严谨感与阅读美感。
- 主要目的：将高数试卷与复习提纲以高密度、可读性强的版式呈现，帮助学生快速浏览题目、对照考点
- 目标用户：高校本科生、考研学生、自学者
- 产品价值：把"刷题"与"看提纲"两件事统一在一份优雅的阅读体验中，提升复习效率与沉浸感

## 2. 核心功能

### 2.2 功能模块
1. **首页（卷宗总览）**：刊头 Hero、试卷档案列表、章节提纲索引、数据统计带
2. **试卷页（真题卷宗）**：试卷封面信息、题型分区（选择/填空/计算/证明）、题目正文与分值、答题卡侧栏
3. **提纲页（考点脉络）**：章节目录、知识点树状结构、考点权重条、典型例题引用

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 刊头 Hero | 期刊式标题、卷期号、主标语、装饰性数学符号 |
| 首页 | 试卷档案 | 按年份/学期排列的试卷卡片，含分值、时长、难度标签 |
| 首页 | 提纲索引 | 章节列表，含考点数量、权重条 |
| 首页 | 数据统计带 | 题目总数、章节总数、覆盖年份等数字展示 |
| 试卷页 | 试卷封面 | 课程名、学期、满分/时长、命题人、注意事项 |
| 试卷页 | 题型分区 | 一/二/三/四大题分区，每题含题干、分值、作答区 |
| 试卷页 | 答题卡侧栏 | 题号网格、进度指示、快速跳转 |
| 提纲页 | 章节目录 | 左侧固定目录，章节编号与标题 |
| 提纲页 | 知识点树 | 章节展开后的知识点列表，含掌握程度标签 |
| 提纲页 | 考点权重 | 横向权重条展示各考点重要程度 |
| 提纲页 | 典型例题 | 关联的典型例题卡片，可跳转试卷 |

## 3. 核心流程

用户进入首页浏览试卷档案与提纲索引 → 选择某份试卷进入试卷页逐题阅读 → 或选择某章节进入提纲页查看考点脉络 → 在提纲页通过典型例题跳回试卷对应题目。

```mermaid
    "试卷页" --> "首页"
    "提纲页" --> "首页"
```

## 4. 用户界面设计

### 4.1 设计风格
- **美学方向**：学术期刊 / 编辑设计（Editorial / Scholarly Journal），融合古典数学教科书的严谨与现代排版的呼吸感
- **主色**：墨黑 `#1a1a1a` + 米白纸色 `#f5f1e8` + 学术红 `#8b1e1e`（强调）+ 墨绿 `#2d4a3e`（次要）
- **字体**：标题用衬线字体（如 "Noto Serif SC" / "Source Han Serif"），正文用宋体/衬线，数字与公式用等宽字体
- **布局**：多栏网格、不对称排版、刊头横线、页眉页脚装饰
- **装饰元素**：数学符号（∫ ∑ ∂ π ∞）、网格底纹、纸张纹理、章节编号圆圈

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 首页 | 刊头 Hero | 衬线大标题、卷期号、双栏副标题、装饰积分符号 |
| 首页 | 试卷档案 | 卡片式列表、年份标签、难度色块、悬停上浮 |
| 试卷页 | 题型分区 | 罗马数字大题号、分值徽章、题干衬线排版 |
| 试卷页 | 答题卡 | 固定侧栏、题号方格、进度条 |
| 提纲页 | 知识点树 | 缩进列表、圆点标记、权重横条 |

### 4.3 响应式
桌面优先（多栏网格），平板自适应为双栏，移动端折叠为单栏并隐藏侧栏答题卡。
