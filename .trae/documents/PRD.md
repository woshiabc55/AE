# 产品需求文档（PRD）

## 1. 产品概述

**产品名称**：Game Asset Reference Hub（游戏素材参考库）

**一句话描述**：一个面向二次创作、设定参考的现代化游戏角色/皮肤/设定图可视化收录工具，支持多游戏分类、标签筛选、详情预览与本地/远程素材管理。

**核心定位**：不直接提供或抓取未经授权的官方素材，而是提供一个可扩展的“容器 + 浏览器”，让用户在合法拥有或已授权的前提下，统一整理、浏览和参考各游戏的公开信息及对应图片资源。

## 2. 目标用户

- 二次创作者（同人画师、模型师、Cosplayer）
- 游戏设定整理者、百科编辑
- 需要跨游戏快速比对角色/皮肤视觉元素的设计师

## 3. 核心功能

### 3.1 游戏与素材浏览
- 支持多游戏收录，按游戏、分类、标签、角色、皮肤维度组织。
- 瀑布流/网格双视图浏览素材缩略图。
- 点击缩略图进入详情页，展示大图、基础信息、标签、备注、来源链接。

### 3.2 分类与筛选
- 左侧边栏展示游戏列表与分类树（按游戏 → 角色/皮肤/设定集/其他）。
- 顶部搜索栏支持角色名、皮肤名、标签关键字搜索。
- 支持标签云筛选、最近添加、收藏夹。

### 3.3 素材来源与数据导入
- 工具本身不内置批量爬取功能。
- 支持通过 JSON 配置文件或拖拽本地图片目录批量导入元数据与图片。
- 支持外部链接（URL）作为图片来源，并在详情页保留来源声明。

### 3.4 收藏与备注
- 用户可对素材收藏、添加备注、评分。
- 收藏与备注状态保存在浏览器本地（localStorage）。

### 3.5 响应式与暗色主题
- 默认暗色主题，符合游戏参考库氛围。
- 适配桌面端、平板端、手机端。

## 4. 数据来源与版权说明

- 工具内置示例数据使用占位图（SVG 生成的抽象色块/图标），不代表任何真实官方素材。
- 真实游戏图片、设定集、角色立绘等版权归各游戏厂商所有。
- 用户自行导入或引用图片时，需确保拥有合法权利或符合官方使用规范。
- 工具鼓励通过“来源链接”字段标注素材出处，便于二次创作者追踪版权信息。

## 5. 设计方向

**整体风格**：暗色赛博档案库（Dark Cyber Archive）。

- **主色调**：深炭黑背景（#0a0a0f）+ 电光青（#00f0ff）+ 霓虹洋红（#ff2a6d）。
- **辅助色**：高对比白（#e0e0e0）、暗灰卡片（#14141c）、半透明边框。
- **字体**：Display 字体使用具有科技感、游戏感的无衬线字体；正文字体使用清晰易读的无衬线字体。
- **氛围**：类似控制室/资料库，带有细微扫描线、光晕、网格背景，营造“正在调阅机密档案”的沉浸感。
- **动效**：页面加载时的矩阵式 stagger 动画、卡片悬停的微放大与光晕、筛选切换的平滑过渡。

## 7. 页面结构

### 7.1 布局
- **顶部导航栏**：Logo、全局搜索、视图切换（网格/瀑布流）、主题切换、导入按钮。
- **左侧边栏**：游戏分类树、快捷入口（全部/收藏/最近添加）。
- **主内容区**：素材卡片网格/瀑布流。
- **右侧详情抽屉**：点击卡片后从右侧滑出，展示大图与元数据。

### 7.2 卡片
- 缩略图区域（默认占位图）。
- 素材名称、所属游戏/角色/分类。
- 标签列表（最多展示 3 个）。
- 收藏按钮。

### 7.3 详情抽屉
- 大图展示区。
- 名称、所属游戏、分类、标签。
- 来源链接（可点击跳转）。
- 备注输入框。
- 收藏开关。

## 8. 交互说明

- **搜索**：输入时实时过滤，防抖 200ms。
- **分类切换**：点击分类后，主内容区淡入切换结果。
- **卡片悬停**：缩略图轻微放大（1.03 倍），显示“查看详情”遮罩。
- **收藏**：点击心形图标切换收藏状态，图标有粒子爆发动效。
- **导入**：点击导入按钮弹出模态框，提供 JSON 配置粘贴区与本地目录拖拽区（浏览器限制下，实际通过文件选择器实现）。

## 9. 性能要求

- 首屏加载不超过 3 秒（在示例数据 500 条以内）。
- 图片懒加载，仅进入视口时加载真实图片。
- 大数据量时使用虚拟滚动或分页。
- 不使用外部重型框架，保持单文件或少量文件结构，便于离线使用。

## 10. 后续扩展

- 支持 PWA 离线访问。
- 支持导出当前库为 JSON/CSV。
- 支持多语言切换（中/英/日）。
- 可选接入用户指定的官方公开 API/Wiki 作为元数据源（不涉及图片抓取）。
