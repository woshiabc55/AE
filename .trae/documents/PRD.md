## 1. 产品概述

「拼豆半面工坊」是一款面向像素艺术爱好者、动画创作者与游戏开发者的网页端拼豆角色创作与骨架动画工具。用户只需绘制角色半面，系统自动镜像生成完整对称形象，再通过网格化拼豆呈现，绑定骨架后可拖拽关节生成可拉动动画，并将图案与动画数据持久化到浏览器数据库中。

- 核心目标：让用户在几分钟内完成「半面绘制 → 拼豆化 → 骨架绑定 → 可拉动动画 → 数据存档」的完整创作闭环
- 目标用户：拼豆手工爱好者、像素风独立游戏开发者、二次元角色动画创作者
- 市场价值：填补「拼豆图案 + 骨架动画」一体化工具的空白，降低 2D 角色动画创作门槛

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 创作者 | 无需注册（本地数据库） | 绘制、镜像、骨架绑定、动画录制、存档管理 |

### 2.2 功能模块
1. **创作工作台**：半面绘制画布、拼豆网格、镜像预览、调色板
2. **骨架绑定面板**：骨骼节点添加、关节拖拽、权重区域设定
3. **动画时间轴**：关键帧录制、可拉动动画播放、循环控制
4. **存档管理面板**：作品列表、保存/加载/导出/删除

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 创作工作台 | 半面绘制画布 | 左半区域绘制，右侧实时镜像预览，支持画笔/橡皮/填充 |
| 创作工作台 | 拼豆网格化 | 将矢量绘制转为 N×N 拼豆网格，每格独立颜色 |
| 创作工作台 | 调色板 | 预设拼豆色卡（48 色）+ 自定义颜色 |
| 骨架绑定面板 | 骨骼节点编辑 | 点击添加关节，拖拽连接成骨骼链 |
| 骨架绑定面板 | 关节拖拽 | 鼠标拖拽关节，拼豆网格按权重变形 |
| 动画时间轴 | 关键帧录制 | 当前姿态保存为关键帧，时间轴可视化 |
| 动画时间轴 | 可拉动动画播放 | 拖拽滑块或点击播放，关键帧之间插值过渡 |
| 存档管理面板 | 作品列表 | 显示所有存档缩略图、名称、修改时间 |
| 存档管理面板 | 导入导出 | JSON 格式导出/导入，便于分享 |

## 3. 核心流程

**主创作流程**：用户进入工作台 → 在左半画布绘制角色半面 → 系统实时镜像生成完整形象 → 点击「拼豆化」转为网格 → 切换到骨架模式添加骨骼节点 → 拖拽关节测试变形 → 切换到动画模式录制关键帧 → 播放可拉动动画 → 保存到数据库。

```mermaid
flowchart TD
    "A[进入工作台]" --> "B[半面绘制]"
    "B" --> "C[实时镜像预览]"
    "C" --> "D[拼豆网格化]"
    "D" --> "E[骨架节点编辑]"
    "E" --> "F[关节拖拽测试]"
    "F" --> "G[关键帧录制]"
    "G" --> "H[可拉动动画播放]"
    "H" --> "I[保存到 IndexedDB]"
    "I" --> "J[存档管理]"
```

**存档加载流程**：用户打开存档面板 → 选择作品 → 从 IndexedDB 读取数据 → 还原画布、骨架、关键帧 → 继续编辑或播放动画。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深墨色背景 `#1a1625` + 暖橙强调色 `#ff6b35` + 薄荷绿辅助色 `#4ecdc4`，营造手作工坊的温暖与像素游戏的复古感
- **次色调**：拼豆色卡 48 色（参考 Hama 拼豆标准色系）
- **按钮风格**：圆角矩形（8px），主按钮带轻微 3D 凸起阴影，工具按钮扁平
- **字体**：标题用 `Press Start 2P`（像素风显示字体），正文用 `JetBrains Mono`（等宽字体增强工程感）
- **布局风格**：左侧工具栏 + 中央画布 + 右侧属性面板的三栏式布局，顶部模式切换 Tab
- **图标风格**：像素风 SVG 图标，与拼豆主题统一

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 创作工作台 | 半面画布 | 深色画布、左半可绘制区、中线虚线、右半镜像预览、网格背景 |
| 创作工作台 | 工具栏 | 画笔/橡皮/填充/吸管/拼豆化按钮，笔刷大小滑块 |
| 创作工作台 | 调色板 | 8×6 色块网格，当前色高亮，自定义色输入框 |
| 骨架绑定面板 | 骨骼编辑器 | 节点圆点、骨骼连线、添加/删除/连接按钮 |
| 动画时间轴 | 时间轴 | 横向轨道、关键帧菱形标记、播放头、播放/暂停/循环按钮 |
| 存档管理面板 | 作品列表 | 缩略图卡片网格、名称、时间、操作按钮 |

### 4.3 响应式
- 桌面优先（≥1280px）：完整三栏布局
- 平板（768-1279px）：工具栏折叠为图标条
- 移动端（<768px）：单栏堆叠，画布优先显示，工具面板抽屉式弹出

### 4.4 3D 场景指引
本项目为 2D 拼豆动画，不涉及 3D 场景。
