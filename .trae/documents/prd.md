## 1. 产品概述

瓷器设计·青花瓷 离线档案提示词生成器 —— 一个以"离线档案"美学风格呈现的Web展示页面，用于展示和对比两套完整的AI图像生成结构化提示词方案。页面本身即为提示词所描述的视觉风格的实体化呈现，用户可在两套方案之间切换浏览、复制提示词文本，并直观感受不同提示词策略对同一主题（Mandelbox分形×青花瓷）的表达差异。

- 目标用户：AI艺术创作者、提示词工程师、视觉设计师
- 核心价值：提供两套可直接用于Midjourney/Stable Diffusion/DALL·E的高质量结构化提示词，同时页面本身作为设计参考

## 2. 核心功能

### 2.1 功能模块
1. **主展示页**：双方案并列/切换展示区、提示词全文展示区、一键复制功能、方案对比摘要
2. **方案详情页**：单方案完整提示词展示、结构化标签导航、参数高亮

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 主展示页 | 方案切换器 | 顶部Tab切换方案A/B，支持并列对比模式 |
| 主展示页 | 提示词展示区 | 完整提示词文本，支持滚动阅读，关键字高亮 |
| 主展示页 | 一键复制 | 点击复制完整提示词到剪贴板，带反馈动画 |
| 主展示页 | 方案对比面板 | 两方案核心参数差异摘要表格 |
| 主展示页 | 预览图区 | 使用提示词生成的示例图片展示（通过API生成） |
| 方案详情页 | 结构导航 | 侧边栏按模块跳转（主体/风格/构图/文字等） |
| 方案详情页 | 参数高亮 | 色彩值、尺寸、透明度等参数特殊样式标注 |

## 3. 核心流程

用户进入主展示页 → 默认展示方案A → 可切换至方案B或并列对比 → 点击复制获取提示词 → 可进入详情页查看结构化分解

```mermaid
flowchart TD
    "进入主展示页" --> "默认展示方案A"
    "默认展示方案A" --> "切换方案B"
    "默认展示方案A" --> "并列对比模式"
    "默认展示方案A" --> "进入方案A详情"
    "切换方案B" --> "进入方案B详情"
    "并列对比模式" --> "复制任一方案"
    "进入方案A详情" --> "结构化浏览"
    "进入方案B详情" --> "结构化浏览"
    "结构化浏览" --> "复制提示词"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：单色灰阶 + 青花蓝(#1a3a6b)作为唯一彩色强调色
- 按钮风格：细边框、纸质感、微阴影
- 字体：等宽字体用于提示词文本，无衬线CJK字体用于中文标题
- 布局：展板式分层拼贴，技术网格背景
- 图标：几何线条图标，lucide-react

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主展示页 | 方案切换器 | Tab按钮、灰阶底色、蓝色激活态 |
| 主展示页 | 提示词展示区 | 等宽字体、纸质感卡片、滚动容器 |
| 主展示页 | 复制按钮 | 蓝色边框按钮、点击后对勾反馈 |
| 主展示页 | 对比面板 | 双栏表格、参数差异高亮 |
| 主展示页 | 预览图区 | 图片卡片、阴影、角括号装饰 |
| 方案详情页 | 结构导航 | 侧边栏锚点列表、当前项蓝色标记 |
| 方案详情页 | 参数高亮 | 色彩值色块、数字加粗、标签胶囊 |

### 4.3 响应式
- 桌面优先设计，平板适配双栏布局，移动端单栏堆叠

### 4.4 3D场景
- 不适用
