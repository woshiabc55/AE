## 1. 产品概述

直观镜头标准表现文档整理工具 — 一款面向光学工程师、质检人员和标准制定者的可视化标准文档管理平台。支持对镜头性能标准进行结构化录入、可视化展示与智能整理，集成 Zulu AI SDK 实现页面内容的 AI 辅助生成。

- 解决镜头标准文档分散、格式不统一、查阅效率低的问题
- 面向光学行业从业者，提供标准化、可视化的文档管理体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 标准管理员 | 邀请注册 | 创建/编辑/发布标准文档，管理分类体系 |
| 工程师 | 邮箱注册 | 浏览、搜索、收藏标准文档，AI 辅助生成页面 |
| 访客 | 无需注册 | 仅浏览已发布的公开标准文档 |

### 2.2 功能模块

1. **仪表盘页面**: 标准文档概览、最近更新、快捷操作入口、AI 生成入口
2. **标准文档列表页**: 分类筛选、搜索、排序、批量操作
3. **标准文档详情页**: 可视化数据展示、参数对比、版本历史
4. **AI 页面生成器**: 通过 Zulu AI SDK 辅助创建标准文档页面
5. **标准参数可视化页**: 镜头性能参数的图表化展示（MTF 曲线、畸变图等）

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|

## 3. 核心流程

### 3.1 标准文档管理流程

用户登录后进入仪表盘，可通过两种方式创建标准文档：手动创建或 AI 辅助生成。手动创建时填写基本信息和参数数据；AI 生成时通过对话描述需求，系统调用 Zulu AI SDK 生成页面内容。创建完成后进入详情页进行可视化展示和编辑，审核通过后发布。

```mermaid
flowchart TD
    "用户登录" --> "进入仪表盘"
    "进入仪表盘" --> "创建方式选择"
    "创建方式选择" --> "手动创建文档"
    "创建方式选择" --> "AI 辅助生成"
    "AI 辅助生成" --> "Zulu AI 对话"
    "Zulu AI 对话" --> "预览与编辑"
    "手动创建文档" --> "填写基本信息"
    "填写基本信息" --> "录入参数数据"
    "预览与编辑" --> "参数可视化展示"
    "录入参数数据" --> "参数可视化展示"
    "参数可视化展示" --> "提交审核"
    "提交审核" --> "审核通过"
    "审核通过" --> "发布文档"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 深蓝 (#0F172A) + 科技青 (#06B6D4)，传达专业、精密的光学行业气质
- **辅助色**: 暖橙 (#F97316) 用于强调操作按钮和重要状态
- **按钮风格**: 圆角矩形 (8px)，微阴影悬浮效果，3D 按压反馈
- **字体**: 标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular，数据使用 JetBrains Mono
- **布局风格**: 侧边导航 + 顶部工具栏，内容区卡片式布局
- **图标风格**: 线性图标 (Lucide)，统一 20px 尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 仪表盘 | 统计卡片 | 深色卡片，渐变边框，数字使用 JetBrains Mono，悬浮微动画 |
| 仪表盘 | 最近更新 | 列表卡片，左侧色条标识状态，右侧时间标签 |
| 仪表盘 | 快捷操作 | 圆形图标按钮，悬浮放大效果，AI 入口使用渐变光效 |
| 标准文档列表 | 筛选栏 | 下拉选择器 + 日期范围 + 搜索框，紧凑排列 |
| 标准文档列表 | 文档表格 | 斑马纹表格，行悬浮高亮，状态标签使用颜色编码 |
| 标准文档详情 | 基本信息区 | 顶部大标题 + 元数据标签组，面包屑导航 |
| 标准文档详情 | 参数可视化 | 深色背景图表区，发光曲线效果，悬浮数据点提示 |
| AI 页面生成器 | 对话界面 | 左侧对话气泡，右侧实时预览面板，底部输入框 |
| AI 页面生成器 | 模板选择 | 网格卡片，悬浮缩放，选中态渐变边框 |
| 参数可视化 | 图表切换 | 顶部标签切换，图表区深色背景，动画过渡 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 平板适配：侧边栏可折叠，表格简化列数
- 移动端：底部导航替代侧边栏，卡片单列布局，触摸优化

### 4.4 3D 场景指导

- 不适用（本项目以 2D 数据可视化为主）
