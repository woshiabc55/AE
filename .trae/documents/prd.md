## 1. 产品概述

C3 是一个创意编码实验平台，面向开发者与数字艺术家，提供脚本编辑、图片色块提取、视频流同步等实验性工具链。平台以开源精神为核心，让用户自由探索代码与视觉的边界。

- 核心目标：构建一个无目的性的创意沙盒，用户可上传素材、编写脚本、实时预览色块与视频同步效果
- 目标用户：创意开发者、数字艺术家、视觉实验爱好者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 创客 | 无需注册，直接使用 | 编辑脚本、上传素材、搜索技能库、预览同步效果 |

### 2.2 功能模块

1. **工作台页面**：代码编辑器、素材上传区、实时预览面板
2. **技能库页面**：开源工具链搜索、脚本模板浏览、PID语言工具集

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 工作台 | 代码编辑器 | 支持PID语言语法高亮的脚本编辑器，实时编译与执行 |
| 工作台 | 素材上传区 | 拖拽上传图片/视频文件，支持批量导入 |
| 工作台 | 色块提取面板 | 从上传图片中提取主色调色块，以网格形式展示 |
| 工作台 | 视频流同步面板 | 视频播放时实时提取当前帧色块，与视频流同步显示 |
| 工作台 | 超频截图工具 | 高频截取视频帧，生成色块序列时间线 |
| 技能库 | 搜索栏 | 关键词搜索开源工具与脚本模板 |
| 技能库 | 工具链列表 | 展示PID语言相关的开源工具集，支持一键导入工作台 |
| 技能库 | 脚本模板 | 预置创意编码脚本模板，可复制到编辑器修改 |

## 3. 核心流程

用户进入工作台 → 上传图片/视频素材 → 编辑器编写或从技能库导入脚本 → 实时预览色块提取效果 → 开启视频流同步 → 超频截图生成色块时间线 → 导出结果


## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深黑底色 (#0a0a0f) + 霓虹绿 (#00ff88) 强调色 + 品红 (#ff0066) 辅助色
- 按钮风格：圆角微光按钮，hover时霓虹发光效果
- 字体：JetBrains Mono（代码区）、Space Grotesk（UI文字）
- 布局风格：三栏分屏布局，左编辑器、中预览、右工具面板
- 图标风格：线性图标，霓虹描边风格
- 整体氛围：赛博朋克终端感，暗色系创意工具

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工作台 | 代码编辑器 | 深色编辑器背景、霓虹绿语法高亮、行号栏、迷你地图 |
| 工作台 | 素材上传区 | 虚线拖拽区、悬浮发光边框、缩略图预览 |
| 工作台 | 色块提取面板 | 色块网格、色值标签、提取动画 |
| 工作台 | 视频流同步面板 | 视频播放器、实时色块条、帧进度指示器 |
| 工作台 | 超频截图工具 | 截图频率滑块、帧缩略图时间线、色块序列 |
| 技能库 | 搜索栏 | 全宽搜索框、实时下拉建议、搜索图标发光 |
| 技能库 | 工具链列表 | 卡片网格、工具图标、标签筛选、导入按钮 |
| 技能库 | 脚本模板 | 代码预览卡片、一键复制按钮、标签分类 |

### 4.3 响应式设计

- 桌面优先设计，三栏分屏布局
- 平板端：编辑器与预览上下堆叠，工具面板折叠为侧边抽屉
- 移动端：单栏切换视图，底部Tab导航
