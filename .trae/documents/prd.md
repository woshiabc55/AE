## 1. 产品概述

NexusCode — 下一代AI驱动的在线编程平台，将智能代码生成、实时协作与沉浸式开发体验融为一体。面向开发者与技术爱好者，通过AI助手实现从想法到代码的极速转化，重新定义编程的交互方式。

- 核心价值：让AI成为编程的"第二大脑"，通过自然语言驱动代码生成、智能补全与错误修复，大幅降低编程门槛并提升开发效率
- 目标用户：全栈开发者、编程学习者、技术团队

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 开发者 | 邮箱/GitHub注册 | 使用全部编辑器与AI功能 |
| 访客 | 无需注册 | 浏览首页、体验基础AI对话 |

### 2.2 功能模块

1. **首页（Landing）**：品牌展示、核心功能介绍、动态粒子背景、CTA引导
2. **工作台（Workspace）**：代码编辑器、AI对话面板、文件管理器、终端输出

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero区域 | 动态代码流背景动画，品牌标语，进入工作台按钮 |
| 首页 | 功能展示区 | 三大核心功能卡片：AI代码生成、智能补全、实时诊断 |
| 首页 | 代码演示区 | 交互式AI编程演示，用户输入自然语言实时生成代码 |
| 首页 | 底部CTA | 渐变背景引导注册/体验 |
| 工作台 | 文件管理器 | 树形文件目录，支持新建/删除/重命名文件 |
| 工作台 | 代码编辑器 | 语法高亮、行号、多标签页、AI内联补全 |
| 工作台 | AI对话面板 | 自然语言对话、代码生成、错误解释、重构建议 |
| 工作台 | 终端面板 | 代码运行输出、错误日志、AI诊断建议 |
| 工作台 | 状态栏 | 语言模式、编码格式、AI状态指示器 |

## 3. 核心流程

用户打开首页，被沉浸式动态背景和AI编程演示吸引，点击进入工作台。在工作台中，用户通过左侧文件管理器组织项目文件，在中央编辑器编写代码，随时通过右侧AI面板用自然语言请求代码生成或问题解答。AI生成的代码可直接插入编辑器，终端面板实时显示运行结果和AI诊断。

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：赛博朋克暗黑风 — 深邃的暗色基底搭配霓虹青绿与琥珀色高光，营造未来感编程环境
- **主色**：`#0A0E17`（深空黑）、`#00F0B5`（霓虹青）、`#FF6B2B`（琥珀橙）
- **辅助色**：`#1A1F2E`（面板灰）、`#2D3548`（边框灰）、`#E0E6ED`（前景白）
- **按钮风格**：圆角微凸起，霓虹青描边发光效果，hover时glow增强
- **字体**：代码区使用 `JetBrains Mono`，UI使用 `Space Grotesk`（标题）+ `DM Sans`（正文）
- **布局**：经典IDE三栏布局，左侧文件树、中央编辑器、右侧AI面板，底部终端
- **图标**：线性描边风格，霓虹青色，2px描边宽度

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero区域 | 全屏暗色背景，代码雨粒子动画，大标题霓虹发光，CTA按钮脉冲动画 |
| 首页 | 功能展示区 | 三列卡片，玻璃态背景，图标霓虹发光，hover浮起+光晕 |
| 首页 | 代码演示区 | 模拟编辑器界面，打字机效果代码生成，语法高亮，AI光标闪烁 |
| 首页 | 底部CTA | 渐变网格背景，大号CTA按钮，粒子扩散动画 |
| 工作台 | 文件管理器 | 深色树形列表，文件图标着色，hover高亮行，右键菜单 |
| 工作台 | 代码编辑器 | 深色编辑区，行号霓虹青，当前行微高亮，AI补全幽灵文本 |
| 工作台 | AI对话面板 | 气泡式对话，AI消息带霓虹边框，代码块语法高亮，插入按钮 |
| 工作台 | 终端面板 | 经典黑底绿字风格，AI诊断用琥珀色标注，可折叠区域 |
| 工作台 | 状态栏 | 底部固定条，左侧语言/编码信息，右侧AI状态脉冲指示灯 |

### 4.3 响应式设计

- 桌面优先设计，工作台在 ≥1024px 宽度下完整展示三栏布局
- 768px-1024px：AI面板转为可收起抽屉，文件管理器折叠为图标栏
- <768px：单栏模式，底部标签切换编辑器/AI/终端

### 4.4 3D场景指引

不适用 — 本项目采用2D界面+粒子动画方案，不涉及3D场景。
