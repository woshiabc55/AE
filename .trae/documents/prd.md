## 1. 产品概述

秒哒推广助手是一款桌面端应用（Web界面 + EXE打包），旨在为百度秒哒平台推广大使提供一个便捷的桌面工具，集成秒哒平台入口、推广数据看板和快捷操作功能，帮助推广人员高效管理推广任务和追踪推广效果。

- 目标用户：百度秒哒推广大使、AI领域内容创作者、社群运营者
- 核心价值：将秒哒平台推广功能聚合到桌面端，提供一站式推广管理体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 推广大使 | 直接使用 | 浏览平台、查看推广数据、快捷分享 |
| 普通用户 | 直接使用 | 浏览秒哒平台、了解产品信息 |

### 2.2 功能模块

1. **主页面**: 秒哒平台内嵌浏览、快捷导航栏、推广入口
2. **推广中心**: 推广数据概览、邀请链接生成、佣金统计
3. **资源中心**: 推广素材下载、活动信息查看、海报生成

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 平台嵌入区 | 通过内嵌浏览器加载秒哒平台完整功能 |
| 主页面 | 侧边导航栏 | 快速切换不同功能模块，显示当前页面标识 |
| 主页面 | 顶部工具栏 | 窗口控制按钮、搜索框、用户头像、通知入口 |
| 推广中心 | 数据看板 | 展示拉新人数、充值佣金、推广排名等核心指标 |
| 推广中心 | 邀请管理 | 生成专属邀请链接/二维码，一键复制分享 |
| 推广中心 | 佣金统计 | 月度佣金汇总、历史记录、结算状态 |
| 资源中心 | 素材库 | 推广海报、宣传文案、活动素材下载 |
| 资源中心 | 活动日历 | 近期线上线下活动信息、报名入口 |

## 3. 核心流程

用户启动应用 → 进入主页面（自动加载秒哒平台）→ 通过侧边栏切换功能模块 → 在推广中心查看数据/生成邀请链接 → 在资源中心获取推广素材 → 分享推广内容

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：科技蓝 (#2563EB) + 活力橙 (#F97316) 作为强调色
- 辅助色：深灰 (#18181B) 背景 + 浅灰 (#F4F4F5) 卡片
- 按钮风格：圆角矩形（8px），主按钮带微妙渐变和悬停动效
- 字体：思源黑体 / Noto Sans SC，标题 20-24px，正文 14-16px
- 布局风格：左侧固定导航栏 + 右侧内容区，卡片式内容展示
- 图标风格：线性图标（Lucide），统一 20px 尺寸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 平台嵌入区 | 全屏iframe，顶部加载进度条 |
| 主页面 | 侧边导航栏 | 深色背景，图标+文字，选中态高亮，宽 240px |
| 主页面 | 顶部工具栏 | 毛玻璃效果，窗口控制按钮，面包屑导航 |
| 推广中心 | 数据看板 | 3列卡片网格，数字大字体，趋势小图标 |
| 推广中心 | 邀请管理 | 居中二维码卡片，下方复制按钮，分享图标组 |
| 资源中心 | 素材库 | 瀑布流卡片布局，悬浮预览，下载按钮 |

### 4.3 响应式

- 桌面优先设计，最小窗口宽度 1024px
- 侧边栏可折叠为图标模式（60px宽）
- 内容区自适应填充剩余空间
