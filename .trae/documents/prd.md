# 《哥窑》主旨解读 PPT — 产品需求文档（PRD）

## 1. 产品概述
- 一份**单页 Web 演示文稿**（HTML + CSS + JS），用于在课堂/汇报场景下解读《哥窑》短片的人物关系（哥哥=张寄、弟弟=张志元）与核心主旨"传承"。
- 目标用户：影视/文学课程的学生与评审、剧本围读参与者、对非遗哥窑与中式叙事美学感兴趣的学习者。
- 价值：以可视化、沉浸式排版呈现"哥=歌"的字源隐喻与"持守/开创"的角色二重唱，让抽象主旨可被一眼读懂。

## 2. 核心功能

### 2.1 角色与场景
- 角色：仅"讲解者"一名（用户）——通过键盘、滚轮、点击翻页键浏览。

### 2.2 功能模块
1. **封面页（Slide 0）**：标题《哥·歌·窑》、副标题、主旨一句话、开始按钮。
2. **总纲页（Slide 1）**：核心主旨与兄弟二人定位对比。
3. **证据链页（Slide 2）**：以左右分栏呈现"哥哥·持守"与"弟弟·开创"对应镜号与台词。
4. **主旨升华页（Slide 3）**：字源"哥=歌"、方印意象、二重唱概念。
5. **结语页（Slide 4）**：古琴/窑火意象收束 + "感谢指正"尾注。
6. **导航条**：右下角浮窗（上一页/下一页/页码指示）。
7. **键盘交互**：←/→/Space/PageUp/PageDown 翻页。
8. **沉浸氛围**：窑火粒子、毛笔笔触背景、淡入式过渡。

### 2.3 页面细节
| 页面 | 模块 | 功能描述 |
|------|------|----------|
| 封面 | 主标题 | 巨型中文标题 + 英文小字"Ge · Song · Kiln" |
| 封面 | 主旨引言 | "哥窑之'哥'，非长幼之序，乃'歌'之古字——窑火如歌，兄弟如线。" |
| 总纲 | 兄弟对比卡 | 两张侧边卡片：左·哥哥张寄（持灯者），右·弟弟张志元（行路人） |
| 证据链 | 镜号引用块 | 列出关键镜号 9/16/45/53/56 配脚本原句 |
| 升华 | 字源图解 | "哥"字拆解为"可+可"配古字"歌"，链接到"窑火如歌" |
| 结语 | 鸣谢 | "非常感谢您的指正" 与 古琴/窑火意象 |

## 3. 核心流程
- 用户打开 `index.html` → 看到封面（窑火光晕+墨色背景）→ 点击"开始解读"或按 → → 进入 Slide 1 → … → Slide 4 末页 → 任意处按 End/重读返回封面。
- 顶部有进度条指示当前页/总页数。

## 4. 用户界面设计

### 4.1 设计风格
- **主题**：东方水墨 + 瓷釉冰裂（宋瓷极简主义）。
- **主色板**：
  - 墨黑 `#0e0d0b`（背景主色）
  - 窑火金 `#c08a3e`（强调/按钮/进度条）
  - 梅子青 `#7faea1`（次强调、哥哥卡片）
  - 紫金土 `#9a3a2a`（弟弟卡片、印章）
  - 瓷白 `#f4ecdc`（正文）
  - 烟青 `#5a6770`（辅助文字）
- **字体**：
  - 中文标题：`Noto Serif SC`（思源宋体）700/900，配合行楷点缀字。
  - 中文正文：`Noto Serif SC` 400/500。
  - 英文/数字：`Cormorant Garamond` 配合 `JetBrains Mono` 用于镜号。
- **按钮/卡片**：圆角 4px，边线 1px 半透明瓷白，长阴影柔光。
- **布局**：16:9 演示比例，桌面优先，向上/向下滚动时锁页。

### 4.2 页面设计概览
| 页面 | 模块 | UI 元素 |
|------|------|----------|
| 封面 | 标题区 | 巨字"哥·歌·窑"，下方"GE·SONG·KILN"小字，底部一条窑火金横线 |
| 总纲 | 对比卡 | 左右两片卡片，中间一条竖向"持守/开创"分隔线 |
| 证据链 | 镜号条 | 每行 `[镜号XX]` 灰底徽标 + 引文 + 引文归属 |
| 升华 | 字源图 | 中央"哥"大字，下方古字"歌"配橙色连接线 |
| 结语 | 鸣谢条 | 顶部窑火粒子渐隐，底部居中"感谢指正" |

### 4.3 响应式
- 桌面优先（≥1280px 最佳展示）。1024px–1280px 自适应。≤768px 简化为上下滑动版本（保留阅读体验）。

### 4.4 3D 场景引导（不适用）
- 本项目为纯排版/动画演示，不涉及 3D / HDRI。
- 改用 CSS 粒子（火星/烟尘）+ SVG 水墨扩散作为氛围层。

## 5. 内容来源与引用
- 镜号与台词均来自用户提供的脚本（第一/二/三幕）。
- 严格遵循"哥哥=张寄、弟弟=张志元"的修正设定。
- 文末保留"非常感谢您的指正"原句作为修订印记。

## 6. 非目标
- 不做账号、不做后端、不做联网。
- 不导出为 .pptx（输出为可在浏览器演示的 HTML）。
