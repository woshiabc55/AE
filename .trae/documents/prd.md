# 哥窑·主旨解读 PPT 项目 PRD

## 1. 项目概述

### 1.1 项目名称
**《哥窑》主旨解读交互式演示文稿**（Ge Kiln · Theme Interpretation Deck）

### 1.2 项目定位
基于用户提供的中文分镜脚本与人物关系修正说明，制作一套**多页交互式 HTML 演示文稿（PPT）**，并配套生成**大量角色/分镜/场景解读 HTML 单页**。整套作品以"哥窑之'哥'——窑火如歌，兄弟如线"为核心主旨，从总纲、人物证据链、主旨升华三个层级进行视觉化阐释。

### 1.3 目标受众
- 影视/广告创意评审组
- 制片方与文化品牌方
- 文化遗产传播者与教育者
- 提案/汇报场合的观众

## 2. 核心主旨（不可变）

> **《哥窑》是关于"传承"的视觉史诗。**
> "哥"通"歌"——窑火如歌，兄弟如线。

### 人物关系（修正后绝对设定）
| 角色 | 设定 | 核心特质 |
|------|------|----------|
| 哥哥 | **张寄** | 守护者、理性、承担（持灯者/锚点） |
| 弟弟 | **张志元** | 探索者、感性、天赋（行路人/延伸） |

## 3. 功能需求

### 3.1 主 PPT（`index.html`）
- 14 张幻灯片，键盘 ← → / 空格 翻页，鼠标点击翻页
- 进度条与当前页码
- 主题切换：水墨青瓷 / 窑火暖金（暗色）
- 配乐按钮（古琴单音、窑火、战鼓、鸟鸣——通过 Web Audio API 合成）

### 3.2 配套 HTML（大量）
| 类别 | 数量 | 文件名规则 |
|------|------|------------|
| 主旨专题页 | 1 | `theme.html` |
| 角色档案页 | 2 | `character-zhangji.html` `character-zhangzhiyuan.html` |
| 三幕分幕页 | 3 | `act-1.html` `act-2.html` `act-3.html` |
| 分镜分镜卡 | 66 | `shot-XX.html`（覆盖镜号 1-66） |
| 主旨解读深度页 | 1 | `essay.html` |
| 资源索引页 | 1 | `gallery.html` |

**总计 74 个 HTML 页面**。

## 5. 设计风格

- **色彩**：墨黑 (#0c0a09)、青瓷釉 (#7ea89a)、窑火金 (#d99a52)、宣纸白 (#f1ebe0)
- **字体**：
  - 中文标题：思源宋体 / Noto Serif SC / 系统宋体
  - 中文正文：思源黑体 / Noto Sans SC
  - 英文/数字：Cormorant Garamond / EB Garamond
- **视觉元素**：
  - 水墨晕染（CSS 径向渐变 + 噪点 SVG）
  - 瓷器开片（CSS 多重 box-shadow 模拟金丝铁线）
  - 窑火（CSS 动画 keyframes 火光）
  - 古琴音（Web Audio 合成单音波形）
- **布局**：12 列栅格 + 大面积负空间 + 不对称重心

## 6. 验收标准

- 所有 HTML 在现代浏览器中无控制台错误
- 键盘与鼠标翻页流畅
- 所有分镜号（1-66）都有对应 HTML 单页
- 配色与字体符合青瓷/窑火美学
- 中文字体回退良好（macOS / Windows / Linux 通用）

## 7. 交付物

- `index.html` —— 主 PPT 入口
- `pages/*.html` —— 配套 73 个 HTML 单页
- `assets/css/main.css` —— 全局样式
- `assets/js/main.js` —— 翻页与音频控制
- `assets/images/*.svg` —— 视觉元素（瓷器、火焰、印章等）
- `README.md` —— 项目说明
