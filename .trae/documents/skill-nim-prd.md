## 1. 产品概述

Skill-Nim —— 一款面向提示词工程师和视觉设计师的在线文档+SVG图形制作工具，集成文本编辑、SVG矢量图形生成、图像拼接合成三大核心能力。用户可在同一界面中编写结构化提示词文档、绘制/生成SVG图形、将多个SVG与图片素材拼合为完整的概念设计板，最终导出为PNG/SVG文件。

- 目标用户：提示词工程师、AI艺术创作者、视觉概念设计师
- 核心价值：将提示词文本化工作流与视觉化工作流统一在同一工具中，消除"写文档→画图→拼板"的跨工具切换成本

## 2. 核心功能

### 2.1 功能模块
1. **文档编辑器**：Markdown/纯文本编辑区，支持结构化提示词模板、语法高亮、字数统计、一键复制
2. **SVG图形工坊**：SVG代码编辑+实时预览，内置青花瓷/离线档案风格SVG模板库，支持参数化生成
3. **拼接画板**：拖拽式画布，可将SVG图形、文本块、图片素材自由排列组合，支持对齐辅助线、图层管理、导出PNG/SVG

### 2.2 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 工具主页 | 三栏布局 | 左栏文档编辑器、中栏SVG工坊、右栏拼接画板，可切换聚焦 |
| 文档编辑器 | Markdown编辑 | 实时预览、模板插入、字数/词数统计、导出TXT/MD |
| SVG工坊 | 代码编辑+预览 | SVG代码编辑、实时渲染、模板库、参数化控件、导出SVG |
| 拼接画板 | 拖拽画布 | 拖入SVG/文本/图片、自由排列、对齐辅助、图层管理、导出PNG |

## 3. 核心流程

用户进入工具主页 → 选择工作模式（文档/SVG/拼接）→ 在文档编辑器中编写提示词 → 在SVG工坊中生成图形 → 在拼接画板中组合为概念设计板 → 导出成品

## 4. 用户界面设计

### 4.1 设计风格
- 延续项目"离线档案"美学：技术网格背景、纸质卡片、角括号装饰、青花蓝强调色
- 工具栏采用深色侧边栏风格，编辑区域为浅色纸质感
- 字体：JetBrains Mono用于代码/编辑区，Noto Sans SC用于中文UI
- 布局：三栏可切换，默认中栏SVG工坊聚焦

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 工具主页 | 模式切换器 | 顶部Tab：文档/SVG/拼接，蓝色激活态 |
| 文档编辑器 | 编辑区 | 等宽字体textarea、行号、实时预览分栏 |
| 文档编辑器 | 工具栏 | 模板插入、字数统计、导出按钮 |
| SVG工坊 | 代码编辑区 | 等宽字体textarea、语法高亮提示 |
| SVG工坊 | 预览区 | 实时SVG渲染、缩放控制 |
| SVG工坊 | 模板库 | 侧边栏模板列表、参数化滑块控件 |
| 拼接画板 | 画布区 | 拖拽画布、辅助线、缩放平移 |
| 拼接画板 | 图层面板 | 右侧图层面板、上下移动/删除 |
| 拼接画板 | 导出栏 | 底部导出按钮、格式选择PNG/SVG |

### 4.3 响应式
- 桌面优先，三栏布局；平板双栏；移动端单栏堆叠

### 4.4 3D场景
- 不适用
