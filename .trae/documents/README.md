# 剧本制作模块 · 文档索引

> 本目录汇集「剧幕 PromptStage」剧本制作模块的完整设计、架构、参考文档。
> 读者：产品 / 设计 / 前端 / 编剧创作者

## 模块边界

剧本制作模块包含三大页面 + 一套共享数据/服务层：
- **首页 / 模板展厅** — 入口与发现
- **剧本编辑器 `/editor`** — 创建与编辑模板
- **变量工坊 `/workshop`** — 填变量生成最终提示词
- **模板详情 `/gallery/:id`** — 浏览与使用入口
- **我的剧库 `/library`** — 云端资产管理

## 设计原则（贯穿所有文档）

1. **结构胜过自由** — 模板是结构化的，变量插槽是声明式的
2. **预览即所得** — 每次输入都立刻看到渲染效果
3. **失败可恢复** — 草稿自动保存，错误可回退
4. **墨黑剧场美学** — 编辑感衬线 + 琥珀金高亮 = 让创作像在排片场
5. **零学习成本** — 模板示例一键套用，无需 prompt engineer 背景
