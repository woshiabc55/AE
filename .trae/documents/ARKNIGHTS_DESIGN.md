# 设计文档 — 明日方舟风格界面

> "在数据与战术的交汇处，每一个像素都像一段作战指令。"

## 1. 产品概述
本项目以《明日方舟》(Arknights) 为视觉蓝本，演示一套**战术 HUD 风格**的网页界面系统。所有视觉决策都围绕"信息密度 + 命令中心感 + 暖色警示"展开。

- 目标受众：喜爱战术 / 科幻 / 工业风的 UI 设计师与游戏玩家
- 核心价值：在浏览器内复刻战术指挥中心的视觉氛围
- 关键差异：硬切角 / 扫描线 / 数据条发光 / Rajdhani 工业字体

## 2. 核心特性

### 2.1 模块
1. **BootScreen 开机画面**：进度条 + 系统加载列表
2. **顶栏 HUD**：HQ 等级、合成玉、原石、系统时间
3. **Tab 导航**：HQ / 干员 / 作战 / 招募 / 商店
4. **干员面板**：左列名单 + 右侧大卡（头像、属性条、档案、引用语）
5. **作战面板**：6 张关卡卡片（代号、难度、理智消耗、掉落、行动按钮）
6. **HQ 面板**：欢迎横幅 + 4 块任务进度 + 派系列表
7. **招募面板**：标签多选 + 稀有度预测 + 招聘按钮
8. **商店面板**：4 件商品（图标、价格、兑换）

### 2.2 交互
- 干员点击切换 → 详情卡片换色 + 进度条更新
- 招募标签点击 → 切换激活态 + 动态重算稀有度预测
- 关卡 open=false 时整卡半透明 + 锁图标
- 顶部 Tab 激活态有 0.5px 发光下划线
- 整页 6 秒循环扫描线

## 3. 核心流程
1. 用户进入 → 触发 BootScreen（0–2 秒进度条）→ 自动进入主页
2. 默认 Tab 为"干员"，可点击切换其它 Tab
3. 干员面板：左侧 6 名干员 → 点击切换右侧详情
4. 招募面板：勾选标签 → 实时显示预测稀有度
5. 任意时刻点 "← EXIT" 回到 Skill Forge 首页

## 4. 用户界面设计

### 4.1 设计风格
- **背景**：`#0E1116`（深空黑）
- **表面层级**：`#181D24` (surface) / `#1F242C` (surface2)
- **边框**：`#2A3340` / `#3A4452`
- **主强调**：`#F5A623`（琥珀橙）
- **次强调**：`#5BA3F5`（青蓝） / `#FFD700`（金） / `#B084EB`（紫）
- **文字**：主 `#E8E8E8` / 弱 `#8A95A5`
- **状态色**：成功 `#5DD39E` / 危险 `#E15A4D`
- **字体**：Rajdhani（工业无衬线） + Noto Sans SC（中文） + JetBrains Mono（数据）
- **形状语言**：硬切角（clip-path 8–12px）、细线分隔、矩形徽章
- **数据条**：发光阴影 + 线性渐变

### 4.2 组件
| 组件 | 描述 |
|------|------|
| `StatBar` | 横向属性条，标签 / 进度 / 数值三段式 |
| `ClassIcon` | 职业符号（⚔ / ✚ / ⊕ / ✦ …） |
| `Star` | 稀有度星级 |
| `BootScreen` | 进度条 + 启动日志 |
| `clip-corners` | 硬切角工具类（切掉右上 12px + 左下 12px） |
| `clip-tag` | 标签形切角（右下一刀） |

### 4.3 响应式
- 桌面优先（1280px+）：双列干员面板
- 平板（768–1280px）：Tab 横向滚动、关卡 2 列
- 手机（<768px）：单列堆叠、底栏 HUD 折叠

## 5. 非功能性要求
- 启动动画 ≤ 2 秒（进度条 50ms tick）
- 所有切角使用 CSS clip-path（避免 border-radius 圆润感）
- 扫描线、脉冲点、发光线全部纯 CSS（不引第三方动画库）
- 字体从 Google Fonts 加载，fallback 到系统字体
