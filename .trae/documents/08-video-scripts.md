# 08 · 使用视频脚本

> 3 段产品使用教学视频。分镜 + 旁白 + 时长 + 后期要点。
> 每段视频 60-90 秒，适配抖音 / 视频号 / B站短视频。

---

## 视频 1 · 「90 秒，从 0 到第一条爆款提示词」

**目标**：让访客 90 秒内完成"打开 → 选模板 → 填变量 → 复制"全流程

**时长**：88 秒
**画幅**：9:16 竖屏
**风格**：节奏快，节拍 0.8s 一刀；屏幕录制 + 旁白 + 字幕

### 分镜

| # | 时长 | 画面 | 旁白 |
|---|------|------|------|

### 旁白脚本（合并版）

```
做内容，每天都在重复写提示词？
同样的结构，写了一遍又一遍。
剧幕让这件事变成：写一次，用一千次。

打开模板展厅，挑一个适合你的。
短视频、种草、直播、小说、分镜、大流量…共 6 大类 16 套模板。
比如这条抖音爆款带货。

点开，4 个 Tab 全是干货。
3 个变量：产品名、目标人群、痛点。
每个模板都附 2 个真实填好的示例。

点立即使用。
左侧填变量，右侧实时出最终提示词。
改一个变量试试。
右侧同步刷新，连 token 数都算好了。

一键复制。
直接粘到 ChatGPT、文心、Kimi 任何地方。

剧幕 PromptStage，让创作像排片场。
免费注册，立即开始。
```

### 后期要点

- 节奏 0.8s 一刀，剪辑点必须卡在鼠标点击 / 页面切换瞬间
- 背景音乐用 110 BPM 的极简电子乐，弱化让旁白突出
- 字幕用 Fraunces 衬线，琥珀金高亮关键词
- 鼠标点击位置用 50% 透明琥珀色圆圈 highlight
- 结尾用 0.4s 黑屏过渡 + Logo

## 视频 2 · 「30 分钟，搭建你的第一个可复用模板」

**目标**：演示编辑器三栏工作流，让有想法的创作者 30 分钟内搭建出第一个模板

**时长**：120 秒（这是 v1 demo 视频，可剪辑为 60s 精华版）

**画幅**：16:9 横屏
**风格**：慢节奏、近景演示 + 旁白 + 字幕 + 步骤卡片

## 通用制作规范

### 1. 屏幕录制

- **分辨率**：1920×1080（横屏）/ 1080×1920（竖屏）
- **DPI**：150% 系统缩放（保证文字清晰）
- **录制工具**：macOS 自带 QuickTime / OBS
- **滚动**：用 SmoothScroll 浏览器插件，禁用鼠标 hover 弹窗

### 2. 旁白

- 工具：Macsay / 专业录音棚
- 声音：中低频磁性男声 / 温柔女声
- 节奏：每分钟 220-240 字（视频 1）/ 180-200 字（视频 2）
- 处理：EQ 切掉 100Hz 以下，加轻微混响

### 3. 字幕

- 字体：思源黑体 / 苹方
- 大小：屏幕宽度的 1/30
- 描边：2px 黑色
- 关键词琥珀金 `#E8B14A` 高亮
- 位置：底部 1/3 安全区

### 4. 背景音乐

- 视频 1 / 3：110 BPM 极简电子乐（情绪轻快）
- 视频 2：90 BPM Lo-fi 钢琴（专注）
- 音量：-18dB（旁白为主）

### 5. 转场

- 默认 0.3s 交叉淡化
- 数字 / 章节切换：0.2s 黑屏
- 鼠标点击：原速无转场，强调"实时感"

### 6. 品牌一致性

- 每次提到产品名「剧幕 PromptStage」
- Logo 在结尾 1.5s + 开场 0.5s 共出现
- 主题色琥珀金 `#E8B14A` 出现在字幕关键词、CTA 按钮
