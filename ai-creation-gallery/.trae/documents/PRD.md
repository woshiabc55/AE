## 1. 产品概述

「AI 二创聚合馆 · Cyber Curatorium」是一个聚合展示抖音热门 AI 二创作品的网页策展画廊。系统按「角色卡牌」「场景壁纸」「物品设计」三大维度收录海量 AI 二创精品，提供沉浸式浏览、多维筛选与鉴赏体验，让分散于短视频平台的 AI 创作灵感得以一站式汇聚。

- 核心目标：将抖音等平台海量、分散的 AI 二创作品按类型系统化聚合策展，提供高品质的灵感浏览与鉴赏入口
- 目标用户：AI 绘画爱好者、二创内容创作者、游戏/插画灵感收集者、卡牌与壁纸收藏党
- 市场价值：解决 AI 二创作品分散难寻、缺乏系统化策展入口的痛点，打造"AI 二创灵感中枢"

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 访客 | 无需注册 | 浏览、筛选、搜索、沉浸查看、收藏至本地 |
| 策展编辑（预留） | 本地标识 | 标记精选、编辑标签（本期仅展示，不开放后台） |

### 2.2 功能模块
1. **聚合首页**：策展宣言 Hero、馆藏数据统计、精选作品轮播、三大分类入口、热门标签云
2. **卡牌馆**：角色卡牌网格、稀有度/风格/阵营筛选、悬浮翻转与光效交互
3. **场景馆**：场景壁纸瀑布流、氛围标签筛选、点击沉浸大图查看
4. **物品馆**：物品设计展示（武器/配饰/道具/载具）、分类筛选、细节卡
5. **作品详情抽屉**：大图、Prompt 标签、作者、热度、相关推荐
6. **全局搜索与筛选栏**：关键词搜索、标签多维筛选、排序（热度/最新）

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|

## 3. 核心流程

**浏览鉴赏流程**：访客进入聚合首页 → 浏览精选轮播与数据统计 → 选择分类入口（卡牌/场景/物品）→ 进入对应馆使用侧栏筛选 → 点击作品打开详情抽屉 → 查看大图与 Prompt 标签 → 浏览相关推荐 → 收藏至本地或返回继续探索。

```mermaid

    "A[进入聚合首页]" --> "B[浏览精选与统计]"
    "B" --> "C[选择分类入口]"
    "C" --> "D{卡牌/场景/物品}"
    "D" --> "E[卡牌馆筛选网格]"
    "D" --> "F[场景馆瀑布流]"
    "D" --> "G[物品馆分类Tab]"
    "E" --> "H[点击作品]"
    "F" --> "H"
    "G" --> "H"
    "H" --> "I[详情抽屉滑出]"
    "I" --> "J[查看大图/Prompt/作者]"
    "J" --> "K[相关推荐]"
    "K" --> "L[收藏或返回探索]"
```

**搜索筛选流程**：访客在顶部搜索栏输入关键词 → 选择标签与排序 → 实时展示跨分类结果网格 → 点击进入详情。

## 4. 用户界面设计

### 4.1 设计风格
- **整体调性**：「赛博策展馆 Cyber Curatorium」——深空墨色为底、霓虹品红与电青双重点缀，玻璃拟态卡片悬浮于光晕之上，营造数字艺术博物馆的沉浸高级感，让色彩绚烂的 AI 作品成为绝对主角
- **主色调**：深墨底 `#0a0a12` / 卡片玻璃 `rgba(255,255,255,0.04)` + 霓虹品红 `#ff2e7e` + 电青 `#22e0d6`
- **次色调**：稀有度色系（普通银 `#9aa3b2` / 稀有蓝 `#4d8dff` / 史诗紫 `#b15cff` / 传说金 `#ffb547`）
- **按钮风格**：胶囊形圆角，主按钮带霓虹辉光投影，次按钮玻璃描边，hover 微光扫过
- **字体**：标题用 `Orbitron`（未来感显示字体，中文配 `Noto Sans SC` 加粗）+ 正文用 `Noto Sans SC`，数字数据用 `Orbitron`
- **布局风格**：顶部固定导航 + 首页全屏 Hero + 分类馆采用左侧筛选栏 + 右侧网格/瀑布流；详情为右侧抽屉
- **图标风格**：线性霓虹描边图标（lucide-react），与赛博主题统一
- **动效**：Hero 光斑缓慢漂移、卡片悬浮 3D 倾斜与扫光、瀑布流懒加载淡入、详情抽屉弹性滑出、数字统计滚动累加

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 聚合首页 | 策展 Hero | 全屏深色、漂浮光斑、大号宣言标题、统计数据条、双入口按钮 |
| 聚合首页 | 精选轮播 | 横向卡片轨道、自动滚动、悬浮放大与信息浮层 |
| 聚合首页 | 分类入口 | 三张大玻璃卡，分类图标+名称+数量，霓虹描边 |
| 卡牌馆 | 卡牌网格 | 卡牌悬浮倾斜、扫光、稀有度描边色、翻转背刻 |
| 场景馆 | 瀑布流 | 错落卡片、氛围渐变叠层、悬浮信息条 |
| 物品馆 | 物品卡 | 居中物品图、底部信息条、稀有度角标 |
| 详情抽屉 | 详情面板 | 右侧抽屉、大图、Prompt 标签芯片、作者条、相关推荐横滑 |

### 4.3 响应式
- 桌面优先（≥1280px）：首页全屏 Hero，分类馆左筛选栏 + 右多列网格，详情右侧抽屉
- 平板（768-1279px）：筛选栏折叠为顶部下拉，网格自适应 2-3 列
- 移动端（<768px）：单列堆叠，Hero 内容缩放，详情抽屉变全屏，瀑布流单列

### 4.4 3D 场景指引
本项目以 2D 图像画廊为主，不涉及 3D 场景；卡牌悬浮采用 CSS 3D transform 营造立体感即可。
