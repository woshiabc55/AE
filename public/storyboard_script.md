## SCENE_01：混沌空间战斗

**时间码：00:00:00:00 - 00:00:01:00（30帧）**

| 帧 | 镜头描述 | 画面内容 | 特效 |
|----|----------|----------|------|
| 0-5 | 特写-呼吸 | 模糊碎片逐渐清晰，战士轮廓显现 | Gaussian Blur从10→0 |
| 5-10 | 全局摇晃 | 镜头剧烈抖动，混沌空间扭曲 | Shake Amount 20→0 |
| 10-20 | 动作合集 | 战士闪现、拳击、打击空爆 | 运动模糊、冲击光效 |
| 20-25 | 马赫运动 | 角色高速移动，残象叠加 | 运动模糊聚集 |
| 25-30 | 超广角 | 凸显角色微小与空间崩坏 | 超广角镜头效果 |

**镜头运镜：**
- 开场模糊碎片 → 聚焦战士
- 跟随角色动作快速切换视角
- 超广角收尾展现空间规模

## 使用说明

1. **运行脚本：** 在Adobe After Effects中打开脚本文件 `project_creation_script.jsx`
2. **生成项目：** 脚本将自动创建所有合成和图层
3. **导入素材：** 将实际的角色、背景素材替换占位图层
4. **调整细节：** 根据实际需求调整关键帧和特效参数

**技术参数对照表：**

| 参数 | 数值 |
|------|------|
| 分辨率 | 1920 × 1080 |
| 像素比 | 1.0 |
| 帧率 | 30fps |
| 场序 | 无场 |
| 颜色深度 | 8位 |
| 音频采样 | 48kHz |

## 特效清单

| 特效名称 | 应用场景 | 参数范围 |
|----------|----------|----------|
| Gaussian Blur | 开场模糊 | 0-10px |
| Shake | 撞击震动 | Amount 0-30 |
| Kaleidoscope | 舞台效果 | Sides: 6 |
| Motion Blur | 高速运动 | Shutter Angle: 180° |
| Glow | 冲击光效 | Intensity: 10-30 |
