# 《窑火不息·追迹》制作类型与 AIGC 创作概念

---

## 一、制作类型定义

### 1.1 作品定位

| 维度 | 定义 |
|:---|:---|
| **作品类型** | AIGC 东方文明叙事短片 |
| **创作范式** | 文本驱动 × 视觉生成 × 声音合成 三合一 |
| **美学范畴** | 缺陷美学 / 窑变美学 / 伤痕美学 |
| **叙事范式** | 破折号停顿美学「主体——动作——了」 |
| **技术路径** | 全流程 AI 辅助创作（非纯 AI 生成） |

### 1.2 与同类作品的差异

| 对比维度 | 传统AI短片 | 本片 |
|:---|:---|:---|
| AI角色 | 工具（指令执行者） | 窑火（不可控变量） |
| 创作模式 | 提示词→生成→满意 | 提示词→生成→接纳"裂纹" |
| 美学标准 | 追求完美/无痕 | 缺陷即神性/伤痕即勋章 |
| 叙事结构 | 线性/因果 | 镜像回环/声音种子 |
| 对话设计 | 自然语言 | 破折号停顿美学 |
| 哲学深度 | 视觉表面 | 四层叙事（表层→心理→哲学→文明） |

## 二、AIGC 创作概念

### 2.1 核心隐喻：AI = 窑火

```
人类创作者（章生一）           AI 生成引擎（窑火）
      │                              │
      │ 拉坯（结构设计）              │ 燃烧（不可控变量）
      │ 分镜（意图）                  │ 生成（偶然性）
      │ 哲学注释（框架）              │ 纹理（超出预期）
      │                              │
      └──────────┬───────────────────┘
                 │
           窑变（共生）
                 │
        哥窑开片（最终成品）
    人类意图 + AI 偶然性 = 不可复制的"活过"
```

### 2.2 人机协作范式

#### 文字层（人类为主）
- 文学剧本：人类创作，AI 辅助润色
- 分镜设计：人类设计，AI 辅助格式化
- 对话范式：人类定义"破折号美学"，AI 辅助校验一致性
- 哲学注释：纯人类产出

#### 图像层（人机协作）
- 提示词工程：人类编写核心提示词，AI 辅助扩展变体
- 图像生成：AI 生成（MJ V6），人类筛选（每镜5-8选1）
- 风格控制：`--style raw` + `--s` 值梯度
- 后处理：人类修复细节，保留"裂纹"

#### 动态层（AI为主）
- 运动生成：Runway/Pika 关键帧动画化
- 相机模拟：推轨/摇镜/升格由 AI 生成
- 转场实现：裂纹溶解/闪白/划像由 AI 合成

#### 声音层（人机协作）
- 旁白 TTS：ElevenLabs 气声生成
- 拟音合成：AI 生成 + 人类调音
- 静默设计：纯人类决策（4镜无对白，3镜纯声音）

#### 剪辑层（人类为主）
- 节奏控制：人类决策
- 镜像结构：人类设计
- 色彩总谱：人类设计 LUT 序列，AI 辅助调色

### 2.3 "缺陷即神性"的 AI 美学

本片的核心命题"真正活过的东西，都会留下裂痕"同样适用于 AIGC 创作方法论：

#### 不追求完美生成
- AI 的"错误"（手指变形、光影异常、纹理错乱）被视为数字时代的"窑变"
- 如同哥窑开片无法精确控制，AI 生成的"裂纹"同样是不可复制的
- 完美的 AI 生成 = 没有活过的瓷器

#### 保留生成痕迹
- 不修掉所有 AI 痕迹，让"裂纹"（生成瑕疵）成为作品的肌理
- 如同金缮/锔钉，修复痕迹本身成为美学
- AI 生成痕迹 = 数字文明的"窑变"记录

#### 破折号美学
- AI 生成之间的"停顿"（加载时间、迭代过程）被纳入叙事节奏
- 如同"宋人烧瓷——"的破折号，AI 生成的间隙 = 时间的悬停
- 人类等待 AI 生成的时刻 = 观众在破折号中的呼吸

## 三、制作流程

### 3.1 流程概览

```
文字层 ──→ 图像层 ──→ 动态层 ──→ 声音层 ──→ 剪辑层 ──→ 调色层
  │          │          │          │          │          │
  │ 文学剧本  │ MJ V6    │ Runway   │ TTS      │ DaVinci  │ LUT序列
  │ 分镜表   │ --s 值   │ Pika     │ 拟音     │ 转场     │ 色彩总谱
  │ 对话范式  │ 25镜     │ 关键帧   │ 环境声   │ 节奏     │ 情感映射
  │          │          │          │          │          │
  └──────────┴──────────┴──────────┴──────────┴──────────┘
                           │
                    反馈迭代循环
                    每阶段对照分镜表审核
```

### 3.2 详细工作流

#### 阶段1：文字层（第1-2周）
1. 文学剧本定稿
2. 分镜表逐镜展开（25镜 × 15项字段）
3. 对话范式校验（12处破折号 + 3处省略号）
4. 哲学注释完整性检查
5. 技术参数表汇总

#### 阶段2：图像层（第2-4周）
1. 为每镜编写 AI 提示词（基于分镜表→AI提示词字段）
2. MJ V6 批量生成（每镜5-8张候选）
3. 人工筛选最佳帧
4. 细节修复（Photoshop/Inpainting）
5. 色彩一致性检查（对照色彩演进总谱）

#### 阶段3：动态层（第4-6周）
1. 关键帧导入 Runway/Pika
2. 设置相机运动参数（推轨/摇镜/升格）
3. 生成动态素材
4. 转场素材制作（裂纹溶解/闪白/划像）
5. 动态质量审查

#### 阶段4：声音层（第3-5周，与动态层并行）
1. 旁白 TTS 录制（ElevenLabs，气声参数调整）
2. 拟音合成（火声/水声/瓷声/雨声）
3. 环境声设计（窑火/展柜/茶寮/雪）
4. 静默设计（4镜无对白，3镜纯声音）
5. 混音（5.1环绕，动态范围控制）

#### 阶段5：剪辑层（第6-8周）
1. 时间线编排（25镜 × 15分钟）
2. 破折号节奏匹配（1.5秒停顿标准）
3. 转场实现（9种转场类型）
4. 镜像结构对应（推泥→推碗，咔→咔）
5. 粗剪→精剪→定剪

#### 阶段6：调色层（第7-8周，与剪辑并行）
1. 11段LUT序列制作
2. 色彩演进总谱套用
3. 情感色彩映射
4. 最终调色输出
