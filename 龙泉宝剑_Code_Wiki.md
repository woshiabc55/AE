## 2. 历史渊源

### 2.1 起源传说

据《越绝书·外传记宝剑》记载，春秋末期，楚王命风胡子到越地寻找欧冶子制造宝剑。欧冶子走遍江南名山大川，寻觅能够出铁英、寒泉和亮石的地方——只有这三样东西都具备，才能铸制出利剑。

最终，欧冶子来到龙泉秦溪山旁，发现：
- 两棵千年松树下有**7口井**，排列如北斗，明净如琉璃，冷澈入骨髓——上等寒泉
- 茨山下蕴藏**铁英**矿石——铸剑原料
- 秦溪山附近有**亮石坑**——磨制宝剑

经两年之久，欧冶子铸成三把名剑：**龙渊、泰阿、工布**。楚王大喜，赐名此宝地为"剑池湖"。

### 2.2 地名由来

- 唐乾元二年（759年），此地置县，以第一把宝剑为县名，称"龙渊县"
- 因避唐高祖李渊名讳，改"龙渊"为"龙泉"
- 龙泉市由此得名，龙渊剑亦从此改称龙泉剑

### 2.3 七星剑之名

秦溪山旁有七口井排列形似天空北斗七星，故龙泉宝剑又称"七星剑"。后世剑匠在剑身上刻七星标志，即为纪念。

## 4. 欧冶子名剑谱

### 4.1 湛卢剑

- **属性**：仁道之剑
- **材质**：铜锡合金，经硫化处理形成黑色菱形花纹
- **主人**：越王允常→吴王阖闾→楚昭王→吴王夫差→越王勾践→薛仁贵→岳飞
- **典故**：吴王阖闾无道，湛卢剑"去之如水"，自行离去至楚。秦王闻而求之不得，兴师击楚
- **诗词**：杜甫"朝士兼戎服，君王按湛卢"

### 4.2 龙渊剑（龙泉剑）

- **属性**：诚信高洁之剑
- **铸剑者**：欧冶子、干将
- **主人**：伍子胥
- **典故**：伍子胥亡命天涯，渔丈人横舟搭救。伍子胥疑其泄密，渔丈人以剑示高洁，横剑自刎
- **备注**：龙渊剑出世之地被楚王赐名"剑池湖"，唐代改"龙渊"为"龙泉"

### 4.3 泰阿剑（太阿剑）

- **属性**：威道之剑
- **长度**：三尺
- **外貌**：橙色，剑身天然镌刻篆体"泰阿"二字
- **主人**：楚王→秦始皇
- **典故**：晋国围楚都三年欲夺泰阿剑。城破之际，楚王拔剑，剑气激射，飞砂走石，旌旗仆地，晋军全军覆没

### 4.4 工布剑

- **属性**：霸道之剑
- **材质**：铁
- **主人**：楚王
- **典故**：后被吴王夫差赐予伍子胥自刎；越灭吴后，又被越王勾践赐予文种自刎

### 4.5 鱼肠剑

- **属性**：勇绝之剑
- **典故**：吴王阖闾使专诸以鱼肠剑藏于炙鱼中，刺杀吴王僚，"披肠夷之甲三事"

## 5. 龙泉宝剑锻制技艺

### 5.1 工艺总览

龙泉宝剑从原料到成品，须经过**28道主要工序**（古法多达90余道）。核心工序为：炼、锻、铲、锉、刻花、嵌铜、冷锻、淬火、磨光等。

### 5.2 剑身锻造工序

根据浙江省地方标准 DB33/T 1375-2024《龙泉宝剑传统制作技术规范》：

**以铁砂为原材料：**

```
配料 → 人工冶炼 → 锻制 → 夹钢 → 砂削（铲、锉）→ 刻图文 → 鎏铜 → 热处理 → 修整 → 磨砺
```

**以钢材为原材料：**

```
锻制 → 夹钢 → 砂削（铲、锉）→ 刻图文 → 鎏铜 → 热处理 → 修整 → 磨砺
```

### 5.3 核心工序详解

#### 一、锤打（锻制）

将铁块在炉中加热烧红，反复锻打成初胚形状。折叠锻造工艺是龙泉宝剑的传统锻制技艺：
- 剑胚出炉后反复折叠锻造，去除原料中的杂质
- 让毛铁和钢融合均匀，提升剑体的韧性和强度
- 剑体厚度通常约8毫米，每折叠1次层数变为原来的2倍
- 最多可达**16000多层**
- 古法锻制，反复多次渗碳工艺锻打，形成宝剑最佳雏形

#### 二、刨锉（砂削）

用钢刀削锉，使剑身厚度适中：
- 剑脊与剑刃之间呈一定坡度
- 剑脊须居剑身正中，并成一直线

#### 三、磨光（磨砺）

将已锉之剑置于厝石（亮石）上磨光：
- 先用粗沙布粗磨
- 后用细沙布细磨
- 再用亮石精磨
- 磨光之工倍于锤打与刨锉，往往需数日甚至数月
- 龙泉北乡特产"亮石"（厝石），剑经亮石磨后不用镀镍，青光照人

#### 四、镶嵌（刻图文/鎏铜）

磨光后在剑身上用钢针镂刻图案、剑名、店号：
- 工匠不用描图，只凭一支钢针在宽仅寸余的剑身两面运针自如
- 镂刻黄龙金凤，栩栩如生；七星如北斗行天，光耀夺目
- 刻好后浇上铜水（赤铜），经铲平加磨
- 青光剑身上呈现金色，产生色彩对比，具有灵光宝气感

#### 五、淬火（热处理）

运用传统淬火之法：
- 将锻好的剑坯烧红，达到700℃～900℃后随即浸入淬火介质中急速冷却
- 淬火后应回火
- "强锻"后使用特殊淬火剂作"柔化"处理
- 使剑身刚柔并寓，能屈能伸
- 属高精工艺，非一般匠工所能得心应手
- 剑质优劣与淬砺的水质、淬火剂、气候等都有很大关系

#### 六、修整

将锻坯或热处理后发生的形变修整到平直：
- 热整：回火时的整剑
- 冷整：其余修整
- 平整

### 5.4 剑鞘制作工序

```
选料 → 开片开槽 → 黏合 → 修正成型
```

### 5.5 剑柄制作工序

```
选料 → 钻孔 → 成型 → 打磨
```

### 5.6 剑装配制工序

```

```

## 6. 四大传统特色

### 6.1 坚韧锋利

- "削铜如泥"
- 1978年，龙泉宝剑制作艺人在全国性集会上当众表演，不费力地将叠在一起的六个铜板劈成两片，剑刃不卷
- 硬剑开刃后能一剑斩开十五、六个铜板而不伤刃

### 6.2 刚柔相济

- 古代用生铁铸造，后来用中碳钢铸造
- 淬火工艺恰到好处，使中碳钢具备弹簧钢特性
- 薄型宝剑可卷成圆圈束在腰中，解开后挺直如故
- 软剑能作360度弯曲，松手如故，又称"腰带剑"

### 6.3 寒光逼人

- 龙泉境内特产"亮石"（厝石）
- 全靠手工磨光，从粗磨、细磨到精磨
- 往往花数日甚至数月
- 磨出后青光耀眼，不用镀镍

### 6.4 纹饰巧致

- 剑身刻有七星标志和飞龙图案
- 工匠一不用彩笔，二不照图样，只用一把钢凿在宽不盈寸的剑身上刻凿
- 刻好后浇上铜水，经铲平加磨，飞龙图案生动自然，永不消失
- 选材考究，有"三斤毛铁半斤钢"之说

## 8. 装饰与纹饰艺术

### 8.1 装具材质

| 部件 | 材质 |
|---|---|
| 木质剑鞘 | 花梨木、龙凤檀、黑檀木等硬木；高档剑鞘包裹鲨鱼皮或制作铜鞘 |
| 金属剑装 | 黄铜、紫铜、白银等，通过铸造、雕刻、鎏金、错金银等工艺制成 |
| 高级剑装 | 镶嵌玉石、宝石 |

### 8.2 装饰工艺

#### 金银错工艺（错金银）

1. 在装具表面绘制所需图案
2. 用凿刀刻出外窄内宽的沟槽
3. 将金、银等薄片锤打入槽中
4. 薄片受挤压紧嵌槽壁，不易脱落
5. 用错石将表面打磨平整

#### 镶嵌工艺

在剑装上嵌入玉石、宝石、琉璃等材料，讲究材质与图案的和谐统一。

#### 雕刻工艺

技法包括刻花、凿花、雕花、冲花等。在铜装或木装上手工雕琢瑞兽、花草、云纹、龙纹等吉祥图样。一套精美雕刻剑装往往需十天至数月手工完成。

#### 鎏金与鎏银工艺

将金或银与水银合成汞齐，涂于装具表面，加热使水银蒸发，金银薄膜附着于表面。常用于宫廷礼仪剑或高级收藏剑。

#### 髹漆与包裹工艺

剑鞘涂以黑漆、红漆等，或包裹鲨鱼皮、彩绸。

### 8.3 纹饰文化寓意

#### 龙纹

龙是中华民族图腾，象征权威、尊贵与力量。龙泉宝剑因地名"龙泉"，尤以龙纹为最经典装饰。

| 时期 | 风格 | 寓意 |
|---|---|---|
| 新石器时代（原龙） | 古朴抽象 | 原始崇拜 |
| 商周时期（夔龙） | 神秘威严 | 常见于青铜器 |
| 秦汉时期（飞龙） | 动感十足 | 升腾与活力 |
| 唐宋时期（行龙） | 飘逸洒脱 | 盛世气象 |
| 明清至今（大龙/祥龙） | 庄重华美 | 吉祥与昌盛 |

剑装上常雕有"睚眦"（龙生九子之一），因其好斗喜杀，象征宝剑的慑人威力。

#### 云纹

象征祥瑞、高远与变幻，寓意宝剑如腾云驾雾。

#### 七星纹

源自秦溪山旁七口如北斗排列的古井，象征天命与指引。

#### 八卦纹

象征天地万物的运行法则，寓意道法自然。

## 9. 剑的结构与部位

### 9.1 结构组成

龙泉宝剑主要由**剑身、剑柄、剑鞘、剑装**四部分组成：

| 部件 | 比喻 | 说明 |
|---|---|---|
| 剑身 | "灵魂" | 核心部分，决定剑的品质 |
| 剑鞘 | "衣袍" | 保护剑身，材质与工艺体现品位 |
| 剑装 | "华饰" | 装饰配件，彰显文化内涵 |

### 9.2 部位名称

```
剑首 ── 剑穗
  │
剑把（剑柄）
  │
剑格（护手）
  │
剑脊 ─────────── 剑身 ─────────── 剑尖
  │                                   │
剑刃                                 剑刃

鞘口 ──
  │
护环 ── 剑鞘
  │
剑标 ──
```

| 部位 | 说明 |
|---|---|
| 剑首 | 剑柄顶端，常雕龙纹或嵌宝石 |
| 剑穗 | 悬挂于剑首的装饰丝穗 |
| 剑把 | 握持部分 |
| 剑格 | 护手，防止滑向剑刃 |
| 剑脊 | 剑身中线隆起部分 |
| 剑刃 | 剑身两侧锋利部分 |
| 剑尖 | 剑身最前端 |
| 鞘口 | 剑鞘入口 |
| 护环 | 剑鞘上的金属环 |
| 剑标 | 剑鞘末端金属件 |

## 10. 龙泉地理与自然资源

龙泉位于浙江省西南部，为春秋时期越国古瓌地，八百里瓯江从山中发源。铸剑所需的四大自然条件：

| 条件 | 龙泉资源 | 作用 |
|---|---|---|
| 铁英（原料） | 山溪中蕴含丰富的铁英沙，随手可得 | 提供铸剑原料 |
| 松炭（燃料） | 大片原始森林，以松木居多 | 冶炼燃料；松炭含松脂，可降低铁的熔点，促进铁矿熔化 |
| 寒泉（淬火） | 秦溪山泉，甘冽清澈 | 淬火用水，增加剑身刚度，不易锈蚀 |
| 亮石（磨砺） | 龙泉北乡特产厝石（亮石） | 磨制剑身，使寒光逼人 |

古人云："天有时，地有利，材有美，工有巧"——四者兼备，方能铸就良剑。

## 11. 历代发展脉络

### 11.1 春秋战国

- 欧冶子在龙泉秦溪山铸成龙渊、泰阿、工布三剑
- 开创铁剑锻造先河

### 11.2 三国两晋

- 龙泉铸剑业受国家管理，专为皇室铸造宝剑
- 南朝梁简文帝《七励》："拭龙渊之雄剑"
- 魏晋时期：由生铁炒炼熟铁，再由熟铁炼成钢铁的工艺普遍使用

### 11.3 唐宋

- 唐代：避高祖李渊讳，"龙渊"改称"龙泉"
- 凡制名剑首称"龙泉"，龙泉成为宝剑代名词
- 北宋：宰相何执中（龙泉人）极力提倡铸剑，剑铺林立，古城有"剑铺街"
- 龙泉宝剑迎来欧冶子时代之后的又一个鼎盛时期

### 11.4 清代

- 先后开设郑义生剑铺、廖太和剑铺、千字号剑铺、万字号剑铺
- 清末民初：千字号、万字号、沈广隆（壬字号）被称为龙泉宝剑三大剑铺
- 沿溪北岸一条街设有7家剑铺

### 11.5 民国

- 民国22年（1933）年产约2000把，总价值8000余银元
- 1924年（一说1930年），全国国术比赛，龙泉宝剑被评为最优秀宝剑
- 抗战时期：手杖剑成为士绅时髦必携品

### 11.6 新中国成立后

- 1956年：个体剑铺组成宝剑生产小组
- 1961年：成立宝剑生产合作社
- 改革开放后：集体、个体剑铺纷纷建立，产品畅销全国及世界20多个地区与国家
- 曾为毛泽东、邓小平、杨尚昆、乔石等领导人及尼克松等外国首脑制作藏剑

## 15. 当代发展与保护

### 15.1 产业现状

- 龙泉宝剑已成为龙泉市主导产业之一
- 产品畅销全国各地和世界20多个地区与国家
- 2022年发布全国首个宝剑及日用刀具产业链标准体系建设指南

### 15.2 濒危状况

- 锻剑工艺现代化对传统技艺造成巨大冲击
- 追求经济效益影响剑的质量
- 锻剑之地环境受到破坏
- 传统锻制技艺处于濒危状态，急需抢救和保护

### 15.3 保护措施

- 2006年列入国家级非遗名录
- 2024年发布地方标准 DB33/T 1375-2024
- 2026年人社部设立"刀剑制作工"国家职业标准
- 《龙泉宝剑志》正式发行，填补国内刀剑类专志空白
- 沈广隆铸剑园被评为国家AAA级旅游景区

### 15.4 市场分化

| 类型 | 开刃状态 | 硬度 | 材质 | 用途 | 价格区间 |
|---|---|---|---|---|---|
| 工艺品型 | 未开刃 | ≤45HRC | 普通铬钢/不锈钢 | 挂墙、送礼、把玩 | 300–3000元 |
| 实用型 | 开刃 | ≥58HRC | 百炼花纹钢/夹钢/覆土烧刃高碳钢 | 武术对练、户外劈砍 | 5000元起 |
