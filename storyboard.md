# 《对决》— 影视 PV 巨像开场分镜脚本

> **画幅**：16:9 IMAX 满画幅
> **时长**：≈ 90s（六幕 × 15s）
> **渲染基线**：3 渲 2 角色 + IMS 粒子模糊 + IMAX 物理光照
> **色调基线**：《沙丘》低饱和、苍凉史诗
> **视觉母题**：圣像崛起 / 水墨国风 / 宇宙神力 / 力量瞬间几何扩散
> **核心冲突**：一个中国老头（农夫装扮） vs 大刀客（西域刀客 / 后期延伸为太空巨兽）

## 全局技术约束

- **画面比例**：严格 16:9 IMAX 满画幅，禁止上下黑边
- **渲染参考**：IMS 粒子 / IMAX 实拍物理光照 / 3 渲 2 角色与现实环境融合
- **画面割裂感**：毁灭 / 安全 / 现实 / 水墨 四象限必须可被一眼分辨
- **运动模糊**：连续 5 拳及关键打击瞬间必须含运动模糊 + 速度线
- **运镜力度**：镜头直接服务于"力"的呈现，不为美而美
- **声音与画面耦合**：心跳 / 嗡鸣 / 编钟 / 龙吟必须与镜头切换严格同步
