# Dialogue Writing

## When To Use
- The user asks for "对白", "台词", "dialogue", "写一段对话", or "润色台词".
- The user wants multiple characters to sound distinct.
- The user needs dialogue that reveals character, advances plot, or creates subtext.

## Input To Gather
- Characters involved and their relationship.
- Scene context and emotional stakes.
- What the scene must accomplish (reveal info, escalate conflict, show attraction, etc.).
- Desired tone (witty, tense, tender, hostile, awkward).
- Any constraints (era, dialect, profession jargon, censorship).

## Writing Rules
- Dialogue should do at least two of: reveal character, advance plot, create conflict, deliver subtext.
- Avoid exposition dumps; break information into dramatic questions and evasions.
- Each character should have a unique vocabulary, rhythm, and sentence length.
- Use silence, interruptions, overlaps, and beats to create realism.
- Subtext should often contradict the literal text.
- Read lines aloud in your head to check rhythm.

## Output Format
```
# 场景对白：{场景名}

## 场景信息
- 地点：
- 时间：
- 人物：
- 核心冲突：
- 潜台词：

## 对白正文
{角色A}
（动作/神态）
台词……

{角色B}
台词……

...

## 设计说明
- 每句台词的功能标注
- 节奏与停顿建议
- 可删减或替换的冗余点
```

## Techniques To Apply
- **反讽式对白**：说反话制造张力
- **打断与省略**：制造真实感与情绪
- **重复与变奏**：强化主题或角色口头禅
- **潜台词分层**：表面谈A，实际谈B

## Tone
- Naturalistic but dramatic; avoid on-the-nose statements unless the style demands it.
