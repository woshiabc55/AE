# Comedy Script Design

## When To Use
- The user asks for "喜剧", "搞笑", "段子", "sitcom", "comedy sketch", "幽默", or "轻喜剧".
- The user wants to write a funny scene, episode, or short script.
- The user needs comedic structure, character dynamics, or gag design.

## Input To Gather
- Comedy subgenre (romantic comedy, dark comedy, sitcom, sketch, satire, farce).
- Target audience and platform (short video, TV, stage, film).
- Tone boundaries (family-friendly, adult, workplace, edgy).
- Main comedic engine (misunderstanding, fish-out-of-water, awkwardness, escalation, satire).
- Characters and their core comic flaws.
- Desired length and number of jokes per page/scene.

## Design Rules
- Comedy must come from character, situation, or escalation—not random jokes.
- Establish a comic premise early and escalate it logically to absurdity.
- Use the rule of three, callbacks, and running gags for rhythm.
- Contrast characters whose flaws collide predictably.
- Build to a climax where the absurdity peaks and is resolved.
- Ensure the punchline has setup; avoid punching down or mean-spirited humor unless intended.
