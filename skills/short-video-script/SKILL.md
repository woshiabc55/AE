# Short Video Script Design

## When To Use
- The user asks for "短视频", "抖音剧本", "快手", "TikTok", "Shorts", "Reels", "15秒", "爆款".
- The user wants high-retention short-form content with a strong opening hook.
- The user needs skits, tutorials, reveals, or emotional micro-stories.

## Input To Gather
- Platform and target duration (15s, 30s, 60s, 90s).
- Genre (comedy, drama, horror, tutorial, prank, emotional, marketing).
- Target audience demographics and interests.
- Core hook / first 3 seconds.
- Call to action or ending goal (follow, comment, buy, share).
- Visual style (talking head, VSL, cinematic, selfie, animation).
- Any branding or product to feature.

## Design Rules
- The first 3 seconds must stop the scroll; state a problem, promise, or visual intrigue.
- Every second should earn attention; remove dead air.
- Use a clear beat structure: Hook → Setup → Twist/Value → CTA.
- Optimize for sound-off viewing with captions and visual clarity.
- End with an open loop, emotional payoff, or clear CTA.
- Design for rewatchability and comment-bait when appropriate.
