# Pacing & Tension Design

## When To Use
- The user asks for "节奏", "张力", "pacing", "tension", "拖沓", "节奏感", "戏剧张力", "松紧".
- The user feels a scene or story is too slow, too fast, or lacks stakes.
- The user wants to build and release tension effectively.

## Input To Gather
- The scene, sequence, or full script to analyze.
- Target emotion (suspense, excitement, dread, romance, relief).
- Genre expectations for pacing.
- Current scene length and number of beats.
- Where the audience should feel confused, informed, worried, or satisfied.

## Design Rules
- Tension arises from unanswered questions and unmet desires.
- Alternate tension and release; never sustain peak intensity indefinitely.
- Use scene length to control pace: short scenes accelerate, long scenes deepen.
- End scenes on a question or unresolved beat to pull the audience forward.
- Delay gratification; give information only when necessary.
- Control the gap between what characters want and what they get.
