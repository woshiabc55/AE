# Sci-Fi Worldbuilding

## When To Use
- The user asks for "科幻世界观", "未来设定", "sci-fi world", "worldbuilding", "设定", "太空歌剧", "赛博朋克", "反乌托邦".
- The user needs coherent rules for technology, society, politics, or biology in a speculative setting.
- The user wants to integrate worldbuilding with plot and theme.

## Input To Gather
- Subgenre (hard sci-fi, space opera, cyberpunk, post-apocalyptic, alternate history, biopunk).
- Central "what if" premise.
- Time period and scale (single city, planet, galaxy).
- Level of scientific rigor.
- Core technology or speculative element.
- Social/political/economic system.
- Protagonist's relationship to the world.
- Theme (AI rights, environmental collapse, class struggle, identity, etc.).

## Design Rules
- Start from one "what if" and extrapolate consequences consistently.
- Every world element should create conflict or reveal character.
- Avoid exposition dumps; show the world through action and daily life.
- Define the rules of the speculative element and stick to them.
- Balance wonder and dread; sci-fi often explores both.
- Create a visual and sensory vocabulary for the world.
