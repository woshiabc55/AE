## 二、场景（地理与空间）

### ① 建木之巅（起点）
- **设定**：上古神树顶端平台
- **氛围**：冷紫调（色温冷端）
- **时间感**：黎明前 / 静谧蓄势
- **功能**：小队集结 / 设备调试 / 出发坪

### ② 紊流海（Transit · 中段空域）
- **设定**：建木与长安之间的高空湍流层
- **云层特征**：
  - 形态：凝固的浪（边缘锐利的体积云）
  - 光照：实时变化方向
  - 光柱：云隙漏下光柱（God Ray）
- **物理**：俯冲时云层撕裂，产生**真空尾迹**
- **速度感**：从静止 → 200 km/h 体感加速

### ③ 长安 · 朱雀门（16）（终点）
- **设定**：宏伟城门（明德门/朱雀门中轴线）
- **大气透视**：远景蒙在**淡金光晕**中
- **色温**：暖金（色温暖端，与建木形成冷→暖反差）
- **镜头出画**：从朱雀门**中轴线**穿入，"如箭矢破空"

### ④ 穿界门（19）（通道）
- **门框材质**：**青铜回纹**（饕餮/云雷纹体系）
- **内部**：扭曲的时空隧道
- **穿越视效**：
  - 色彩分离：RGB 错位 **0.3 秒**
  - 声学特征：低频"嗡"声 **50 Hz**
- **出画方向**：门开向东方，门内映射出长安城门（16）轮廓

## 三、道具（可交互物件）

### ① 能量核心（蚩奼持）
- **形态**：掌中/臂载的圆形或卵形装置
- **光色**：**冷蓝**
- **节奏**：明灭频率 = 操作者呼吸频率
- **功能**：提供穿越动力的核心源

### ② 机关翼（伏鹤持）
- **形态**：可展开的金属翼面（机关术外骨骼）
- **展开动作**：弹簧/连杆释放 → 翼面瞬间张开
- **力学**：翼展时使用者需**后仰+重心后移**以抵消前向反作用力

### ③ 穿界门（19）
- **静态特征**：青铜回纹门框 + 内部时空隧道
- **动态特征**：
  - **空间门涟漪**：水波状扭曲，频率 **2 Hz**，振幅随距离衰减
  - **防护结界**：淡金色**蜂窝状网格**，仅在**碰撞时显形**（日常不可见）

## 四、视觉特效（镜头语言）

### 色彩与光
| 段 | 场景 | 色温 | 主光 | 备注 |
|---|---|---|---|---|
| A | 建木集结 | 冷紫 | 顶光冷调 | 高对比、低饱和 |

| C | 朱雀门出画 | 暖金 | 侧逆光 | 淡金光晕大气透视 |

### 运镜
1. **静帧起手**：建木之巅五人楔形站位（俯视/平视混合）
2. **推镜**：能量核心特写（蓝光明灭）
3. **甩镜**：机关翼展开瞬间
4. **跟拍俯冲**：从静止加速至 200 km/h
5. **穿越主观镜头**：进入穿界门 → RGB 错位 → 色彩归位
6. **破空出画**：朱雀门中轴线 → 黑场/下一场

### 物理与速度
- **俯冲加速曲线**：0 → 200 km/h（体感，非客观）
- **真空尾迹**：云层撕裂瞬间产生（空气密度骤变）
- **大气透视**：朱雀门远景淡金光晕（色散+雾效叠加）

### 防护/碰撞系统
- **防护结界**：淡金色蜂窝网格
- **触发逻辑**：仅在物体碰撞时显形（默认隐形，省渲染）

### 3. 技术要点与解法
| 难点 | 实施方案 |
|---|---|
| 0→200 km/h 体感加速 | 速度曲线非线性（前 30% 时长完成 70% 加速），配风噪渐入 |
| 真空尾迹 | 紊流海体积云使用粒子置换 + 低密度区域 mask 反向 |
| RGB 错位 0.3 s | 后期调色：R/G/B 通道分别做 0.3 s 时序偏移（红→绿→蓝）|
| 2 Hz 空间涟漪 | 屏幕空间扭曲：正弦波驱动 UV，振幅按距离衰减 |
| 防护结界显形 | 物理碰撞事件触发材质 alpha，关键帧标记 0/1 切换 |
| 色温渐变（冷紫→暖金）| 时间轴绑定色温参数，关键帧 A=7500K（建木）→ B=3200K（朱雀门）|

### 4. 软件栈建议
- **角色/动画**：Maya / Blender
- **体积云/渲染**：Houdini + Karma，或 Unreal Engine 5（Niagara + Lumen）
- **合成**：Nuke / After Effects（错位、色彩、合成）
- **音频**：Pro Tools / Reaper

### 5. 验收标准（Quality Bar）
- [ ] 电影级材质（皮肤次表面散射 + 金属 IBL）
- [ ] 体积云光照方向实时变化
- [ ] RGB 错位精确 0.3 s（不超不欠）
- [ ] 50 Hz 低频嗡声清晰可感（不破音）
- [ ] 楔形站位 + 队长在前符合分镜
- [ ] 朱雀门出画方向为"中轴线穿入"
- [ ] 图 1 材质基准已贯彻到所有 CG 资产
