## 一、2025年度官方榜单热梗

### 国家语言资源监测与研究中心「2025年度十大网络流行语」

| 热梗 | 含义 |
|------|------|
| **DeepSeek（深度求索）** | 中国本土大语言模型，引发全球关注 |
| **敬自己一杯** | 对努力与平凡的接纳，年轻人缓解压力、实现自我和解的温柔表达 |
| **助我破鼎** | 源自电影《哪吒之魔童闹海》，为自己鼓劲打气，面对困难不放弃 |
| **××基础，××不基础** | 源于穿搭博主"下身基础，上身就不基础"，用反转强调反差或突出部分 |
| **千百次练习只为这一刻** | 强调从量变到质变的学习规律 |
| **情绪价值** | 从营销学延伸到人际关系、职场，指给人带来的情绪满足 |
| **如何呢又能怎** | 出自改编歌曲歌词，以打趣式轻松语气表达豁达态度 |
| **村咖** | 开在乡村里的特色咖啡馆，融合乡村文化与城市消费习惯 |
| **来财** | 源自歌曲重复副歌"来财，来"，表达对好运的期待 |
| **浪浪山小妖怪** | 同名动画电影，以中式美学登顶中国二维动画票房冠军 |

| 热梗 | 含义 |
|------|------|
| **主理人** | 从潮牌圈扩展到泛商业领域，指能将自身特质与品牌强结合的主要负责人 |
| **邪修** | 源自修仙小说，形容离谱但高效的方法和手段，如"邪修做饭""邪修抢票" |
| **魔丸 & 灵珠** | 出自《哪吒》系列，"魔丸"指特立独行，"灵珠"指乖巧理性 |
| **高雅人士** | 源自企鹅魔性舞步梗，反讽在情绪崩溃时仍努力维持外在得体 |
| **做完你的做你的** | 源于景德镇"鸡排主理人"名场面，描述忙碌中从容不迫 |
| **误闯天家** | 以古风意象包装"跨界闯入"的窘迫，自嘲理想与现实的落差 |
| **从从容容、游刃有余，匆匆忙忙、连滚带爬** | 对理想状态与现实窘境的反差表达 |
| **当个事儿办** | 郑重回应安排的表达 |
| **拼好×** | 拼单类消费文化的衍生表达 |

### 《咬文嚼字》「2025十大流行语」

| 热梗 | 含义 |
|------|------|
| **韧性** | 社会集体应对挑战的坚韧特质 |
| **具身智能** | 有物理载体的智能体（如人形机器人），与离身智能（如大语言模型）对应 |
| **苏超** | "江苏省城市足球联赛"简称，因接地气而火爆出圈 |
| **赛博对账** | 双方拿着各自"生活账本"核对数据，消除误解与偏见 |
| **数字游民** | 依托信息技术远程办公，打破地理限制自由流动的年轻人 |
| **谷子** | 英文"goods"谐音，二次元圈代指动漫、游戏周边产品 |
| **预制××** | 如"预制朋友圈""预制文案"，指提前准备好的内容 |
| **活人感** | 形容角色、偶像或虚拟形象真实不做作，像真人一样有温度和个性 |

## 二、2025年全网爆火热梗

| 热梗 | 出处 | 含义 |
|------|------|------|
| **丝瓜汤** | 博主"累子"家庭情景短剧 | 东亚代际矛盾的隐喻，"你肝火太旺，喝点丝瓜汤吧"指代"为你好"式沟通 |
| **回！答！我！** | B站主播"小明剑魔"直播间咆哮 | 魔性咆哮体，配上历史人物动画，用于抽象宣泄 |
| **信××还是信我是秦始皇** | 早期诈骗短信复活 | 调侃不靠谱的说法或自吹自擂的声音 |
| **技能五子棋** | 《喜人奇妙夜2》 | 用无厘头对抗"竞技至上"的反叛情绪 |
| **老登电影** | 网友对姜文《你行！你上！》的吐槽 | 对"爹味"叙事的祛魅，批判套路化英雄叙事 |
| **北京孩子** | 民谣歌手马頔综艺言论 | 文艺青年"清高"的自嘲，既得主流认可又表示不屑 |
| **洪玄烨** | 网络"清朝野史"段子 | 对传统严肃话题的吐槽和叛逆式解构 |
| **馕言文** | 新疆博主短视频 | 夸张倒装句式+新疆口音，如"今天馕吃得多得很嘛" |
| **连滚带爬** | 歌手郑智化吐槽机场 | 吐槽自身无助与狼狈，表达对现实的不满 |
| **法拉利老了还是法拉利** | 拉丁舞裁判谢尔盖·辛金斯 | 消解衰老焦虑，经典永不过时 |
| **City不City** | 外国博主来华旅游视频 | 问某事物是否够"洋气""都市化"，中英混杂玩法 |
| **夹角打工人** | 职场吐槽衍生 | 卷不动又躺不平，卡在中间进退两难 |
| **养龙虾** | 打工人自创AI办公趣味叫法 | 借助AI工具高效完成工作，戏称AI为"龙虾" |
| **外耗** | 心理科普视频 | 停止内耗，合理向外释放情绪，不委屈自己 |
| **床腐** | 年轻人周末居家状态 | 周末全天窝在床上，彻底放松躺平 |
| **别太荒谬** | 短视频口语化吐槽 | 温和吐槽离谱的人或事，不带攻击性 |

## 三、2024年度十大流行语（《咬文嚼字》）

| 热梗 | 含义 |
|------|------|
| **数智化** | 数字化和智能化的融合体 |
| **智能向善** | 人工智能发展必须有利于增进人类福祉 |
| **未来产业** | 由前沿技术驱动的前瞻性新兴产业 |
| **City不City** | 美国博主带火的中英混杂表达，问是否够"洋气" |
| **硬控** | 从游戏术语扩展到社交，形容某事物强烈吸引人 |
| **水灵灵地** | 形容某种行为生动、鲜活或值得称道，也可用于调侃自嘲 |
| **班味** | 因工作压力大表现出的眼神疲惫、面容憔悴等状态 |
| **松弛感** | 面对压力时从容应对、不慌张、不焦虑的心理状态 |
| **银发力量** | 老年群体在社会各领域产生的不可忽视的力量 |
| **小孩哥/小孩姐** | 在某方面有过人才能的孩子，年纪虽小却拥有非凡才能 |

## 五、2026年新兴热梗

| 热梗 | 含义 |
|------|------|
| **我去不早说** | 讽刺对方说的信息人尽皆知，假装因此错过机会 |
| **我将全职在家研究…** | 夸张表达对某事的极度认可/震惊，用"研究"包装想摆烂的状态 |
| **不讲不讲** | 表示不想聊了、不解释了，带点不耐烦的撒娇式结束话题 |
| **此人的XXX恐怕在我之上** | 一本正经地夸张评价，表面谦虚实则阴阳怪气或搞笑吹捧 |
| **DLSS 5 ON** | 游戏圈热梗，把同一人物做成开/关对照图，吐槽AI滤镜/美颜效果 |
| **不知道，我的身材很曼妙** | 电商评论区答非所问神回复，变成"已读乱回"万能句式 |
| **爱你老己** | 自我关怀的表达，对自己好一点 |
| **活人感** | 形容真实不做作、有温度和个性 |
| **在中国我们不说…** | 文化碰撞类梗，强调中国特有的表达方式 |
