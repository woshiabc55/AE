关于本资料库

数据 / 设计 / 实现

IP.ARC 是一个面向玩家、收藏者、ACG 行业研究者的「游戏 IP 衍生作品」综合浏览资料库。

数据来源

资料库由确定性算法根据截至 2026.06.08 已公开的 IP 衍生信息聚合生成，覆盖 297+ 个全球知名游戏 IP 与 67 种衍生作品类型，包含已完结、连载中、制作中、计划中、限定、已下架等所有状态。

使用方式

  • 支持中文/英文模糊搜索
  • 多维筛选：类型、IP、地区、年份、状态、评分
  • 三种视图：网格 / 列表 / 表格
  • 详情面板：完整元信息 + 简介 + 关联推荐
  • 数据可视化：类型分布、年度趋势、地区分布、评分分布
  • 数据导出：当前筛选结果一键导出为 JSON

设计理念

致敬 ACG 文化与复古游戏机柜美学。赛博朋克配色 + 霓虹辉光 + 像素噪点纹理 + CRT 扫描线效果，平衡未来感与怀旧温度。

技术栈

纯静态 HTML + CSS + 原生 ES6+，无构建依赖。所有数据 1.8 MB，加载即可使用。字体来自 Google Fonts。
