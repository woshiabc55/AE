## 二、模块详解

### ① 全局音乐效果（GlobalMusic）
- **环境音调（插值）**：随场景动态过渡音高
- **基准幅度效果**：统一的响度基准曲线
- **简化（简）**：去除冗余音轨，仅保留关键声部
- **输出**：作为底层时间轴，影响所有分镜

### ② 视觉纹理与风格（Texture & Style）
- **幻想纹理**：仙侠 / 玄幻题材的粒子、光晕、墨迹
- **色调严格《--》低饱和**：色彩严格限制在低饱和区间
- **朝向控制**：纹理的流动方向（顺 / 逆 / 双向）
- **用途**：奠定画面整体视觉基调

### ③ 提示词剧本重构（Prompt Reconstruct）
- **直观镜头描述**：把抽象词转化为具象分镜
- **剧本切片**：将长提示词拆分为可独立推理的镜头单元
- **提效策略**：复用 prompt 片段，减少 token 冗余
- **视觉展开（分镜）**：将剧本映射为连续分镜

### ④ 视觉耦合与基准效果（Visual Coupling）
- **音效 ↔ 画面耦合**：打击点 ↔ 视觉冲击同步
- **基准效果**：所有特效的强度基准线
- **全局影响**：音效变化驱动画面参数微调
- **作用**：实现"声画一体"

### ⑤ 物理概念与瞬间打斗（Physics & Combat）
- **物理概念**：惯性、动量、冲量等真实物理量
- **瞬间打斗**：0.1~0.3s 的爆发性动作窗口
- **基础画面**：打斗前的蓄势帧
- **视觉定向**：受力方向、位移方向清晰可读

### ⑥ 画面效果与空间感（Visual FX & Space）
- **瞬间发力 → 变化 → 冲击力 → 毁灭**：递进式视觉链
- **冲击波几何倍扩散**：径向 / 锥形扩散模型
- **画面瞬间空间**：空间瞬时畸变（扭曲、撕裂）
- **空间割裂感**：使用镜头畸变 + 墨迹飞溅强化

### ⑦ 运镜与镜头语言（Camera & Lens）
- **运镜展现力度**：
  - 角色的水墨画打击 + 瞬间线条
  - 镜头瞬间变化（推 / 拉 / 摇 / 移）
- **镜头直观**：观众第一眼即可读懂动作
- **模糊效果**：动态模糊强化速度感
- **画面效果**：光斑、墨点、残影叠加

## 四、提示词模板（Template）

```text
[全局音乐效果]
  配合-环境音调（插值）| 基准幅度效果 | 简

[视觉纹理]
  幻想纹理 + 色调严格《--》低饱和

[提示词剧本]
  直观-镜头 + 分镜展开（ocind视觉展开）

[视觉耦合]
  纹-朝向 + 基准效果 + 跟随全局影响（音效）

[物理与打斗]
  物理概念 + 瞬间打斗 + 视觉定向
  └─ 瞬间-发力-变化-冲击力-毁灭
     └─ 冲击波几何倍扩散

[画面效果]
  画面瞬间空间 + 空间割裂感

[运镜]
  运镜展现力度 + 水墨画打击 + 瞬间线条
  镜头瞬间变化 + 模糊效果

[故事]
  展开故事
```
