About · 关于本项目

哥窑 · 6 段结构 修订版

本项目以短片《哥窑》剧本为核心，输出两类内容：(1) HTML 形态的 11 页 PPT 演示文稿；(2) 覆盖 62 镜的分镜长卷与单镜详情页。 项目采用修订版的兄弟关系——哥哥=张寄，弟弟=张志元，并按 前幕·制作组自白 → 第一~四章 → 尾幕·展开 PPT 的 6 段结构组织内容。

创作说明

  • 剧本来源：用户提供的《哥窑》分镜表，按 6 段结构（前幕 / 第一~四章 / 尾幕）重新组织。
  • 姓名修正：原始脚本中存在"章生一""张志耀""张寄"等混用，本项目以修订版"哥哥=张寄，弟弟=张志元"为绝对准则。
  • 图像生成：每一镜都附有 SDXL 风格的英文提示词，可通过 text_to_image 接口生成；详见 data/shots.json。
  • 技术栈：纯 HTML + CSS + 原生 JavaScript，零依赖，零构建。

关键技术点

  • 数据驱动：所有镜号存于 data/shots.json，6 段结构存于 data/structure.json，页面通过 fetch 加载并渲染。
  • 图像生成：每镜都有 prompt 字段，可调用 text_to_image 生成对应缩略图。
  • 翻页交互：PPT 页支持键盘 ← / → / Space / Home / End 翻页，全屏按 F。
  • 响应式：桌面 1440px 设计，平板/移动端单列自适应。
  • 零依赖：不引入 React/Vue/Tailwind，原生 HTML+CSS+JS。
