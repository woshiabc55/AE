哥窑 · 修订版分镜与主旨解读

本项目以短片《哥窑》剧本为核心，输出两类内容：(1) HTML 形态的 PPT 演示文稿；(2) 覆盖 36+ 镜号的分镜长卷与单镜详情页。 项目采用修订版的兄弟关系——哥哥=张寄，弟弟=张志元。

创作说明

  • 剧本来源：用户提供的《哥窑》分镜表，涵盖序幕与三个主要幕。
  • 姓名修正：原始脚本中存在"章生一""张志耀""张寄"等混用，本项目以修订版"哥哥=张寄，弟弟=张志元"为绝对准则。
  • 图像生成：每一镜都附有 SDXL 风格的英文提示词，可通过 text_to_image 接口生成；详见 data/shots.json。
  • 技术栈：纯 HTML + CSS + 原生 JavaScript，零依赖，零构建。

四幕概览

  • 序幕·现代（镜 1–5）—— 当代送别，引出"哥窑的'哥'是谁"。
  • 第一幕·历史回溯（镜 6–33）—— 战乱南迁、发现紫金土、依山建龙窑、第一窑开火。
  • 第二幕·贡品之印（镜 36–47）—— 朝廷查验、赐"龙泉章氏"贡印、山民欢呼。
  • 第三幕·窑火成瓷（镜 48–66）—— 火候失控、兄弟守候、金丝铁线开片、两个时代背影重叠。

关键技术点

  • 数据驱动：所有镜号存于 data/shots.json，页面通过 fetch 加载并渲染。
  • 图像生成：每镜都有 prompt 字段，可调用 text_to_image 生成对应缩略图。
  • 翻页交互：PPT 页支持键盘 ← / → / Space / Home / End 翻页，全屏按 F。
  • 响应式：桌面 1440px 设计，平板/移动端单列自适应。
  • 零依赖：不引入 React/Vue/Tailwind，原生 HTML+CSS+JS。
