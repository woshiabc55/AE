# STELLARIS · 万象天文台

一个包含 100,000+ 个程序化生成 UI 微元素与现代化强化交互的独立艺术网站。

## 12 个展厅

| #   | 展厅         | 元素数      | 简介                        |
|-----|--------------|-------------|-----------------------------|
| 00  | 序厅         | 1,200       | 字符倾泻的入场仪式         |
| 01  | 星图         | 100,000     | 四臂螺旋星系可缩放星图     |
| 02  | 字海         | 4,096       | 可拖拽字符方块诗阵         |
| 03  | 晶阵         | 8,000       | WebGL 晶柱引力场           |
| 04  | 脉搏         | 960         | 12 圈音频脉冲麦克风可视化 |
| 05  | 样本柜       | 10,000      | 10 类 UI 微缩切片         |
| 06  | 回声廊       | 200         | CSS 3D 无限镜像长廊        |
| 07  | 色谱         | 1,800       | 360° 环形色板可刮图案     |
| 08  | 数据林       | 3,200       | KPI 数字长成的发光森林     |
| 09  | 沙盘         | 1,024       | 1,024 颗粒子 + 9 种效果   |
| 10  | 宣言         | 12,000      | 逐字点亮可朗读长卷         |
| 11  | 尾声         | 1,000       | 1,000 个呼吸光点           |

合计 ~141,000 个微元素。

## 操作

- `↑ / ↓` 或 `PageUp / PageDown`：切换展厅
- `1–9 / 0`：跳转到对应编号的展厅
- `R`：回到序厅
- `M`：静音 / 取消静音
- `F`：进入 / 退出全屏
- 鼠标：磁性光标 + 悬停反馈
- 左侧目录按钮：打开展厅快速导航

## 开发

```bash
pnpm install
pnpm dev      # http://localhost:5173
pnpm build
pnpm preview
```

## 技术栈

- React 18 + TypeScript + Vite 5
- Tailwind CSS 3 + CSS Variables（设计 Token）
- Three.js + @react-three/fiber + @react-three/postprocessing
- Framer Motion（章节切换）
- Zustand（全局状态）
- Web Audio API（麦克风 + 合成）
- Canvas 2D / SVG（2D 元素绘制）
