# NEON.FRAME // 游戏 IP 衍生作品资料库

> 全球首个开源、现代化的游戏 IP 衍生作品浏览平台。聚合 4000+ 款基于游戏 IP 改编/衍生的动画、漫画、电影、电视剧、小说、舞台剧、手办、商品、漫画单行本等条目，覆盖 355+ 主流游戏 IP，横跨 40 余年游戏文化史。

## ✨ 特性
- **海量数据**：355+ 游戏 IP，4000+ 衍生作品条目
- **多维检索**：按 IP、类型、年份、地区、标签、关键词精准筛选
- **现代化设计**：Cyber/Synthwave 暗黑 + 霓虹辉光，桌面优先
- **数据看板**：类型分布、地区分布、年份趋势、Top 10 IP 可视化
- **响应式**：桌面、平板、移动端自适应

## 🛠 技术栈
- React 18 + TypeScript + Vite
- TailwindCSS v3 + 自定义霓虹主题
- Zustand 状态管理
- lucide-react 图标
- react-router-dom 路由

## 📦 数据规模
- **355** 款游戏 IP
- **4,231** 款衍生作品
- **10** 种衍生类型（动画/电影/漫画/小说/舞台剧/手办/周边/音乐/手游/真人剧）
- **20+** 个地区 / 国家
- **40+** 年时间跨度（1985 - 2026）

## 🚀 启动
```bash
npm install
npm run dev   # 开发模式
npm run build # 生产构建
npm run preview
```

## 📁 结构
```
src/
  components/    # 通用组件
  data/          # 静态数据（IP 池 + 衍生作品）
  pages/         # 页面（Home / Browse / Dashboard / About）
  store/         # Zustand store
  utils/         # 工具函数
  hooks/         # 自定义 Hooks
```
