## 9. 协作与同步（坚果云）

本项目通过**坚果云**（Nutstore）实现本地与远程仓库的同步连接：

| 环节 | 说明 |
|------|------|
| 本地开发 | 在坚果云本地同步目录中进行代码编写 |
| 版本控制 | 使用 Git 进行本地版本管理 |
| 远程同步 | 坚果云自动同步本地文件到云端，通过 GitHub 进行远程协作 |
| 远程仓库 | `origin` 指向 `https://github.com/woshiabc55/AE` |

### 同步流程

```
本地编辑 → 坚果云同步 → Git 暂存 & 提交 → Push 至 GitHub
```

### 开发工作流建议

1. **本地开发**：在坚果云同步目录中编辑代码
2. **版本记录**：`git add` → `git commit` 记录变更
3. **远程推送**：`git push origin main` 推送到 GitHub
4. **协作同步**：团队成员通过 GitHub 拉取最新代码
