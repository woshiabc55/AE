技术规格

通信协议

所有模块通过 EventBus 通信，支持 on/off/emit/once。V5扩展通配符与事件拦截。模块间零直接引用。

渲染管线

V1单效果 → V2链式管道(ImageData传递) → V3多图层合成(globalCompositeOperation) → V4/V5保持图层架构

效果接口

标准接口: init(bus) render(ctx, sourceData, time) getParams() setParam(name, value)。V5通过Plugin API暴露。

动画系统

requestAnimationFrame 驱动，CoreModule管理循环，每帧发射 animation:frame 事件携带 elapsed 时间。

UI框架

纯HTML/CSS/JS，零依赖。CRT终端美学，扫描线+暗角+霓虹发光。Press Start 2P + Share Tech Mono字体。

图像处理

Canvas 2D API，ImageData像素级操作。像素化=区域均值采样，离子化=粒子物理模拟，波浪纹=行级正弦位移。
