COMPONENT LIBRARY · 哥窑设计系统

组件库 · 14 个可复用单元

哥窑组件库是一套主题鲜明、语义清晰的设计系统。 每一个组件都源自《哥窑》主旨解读项目的实际使用场景， 并提炼为可独立调用、可组合搭配的 CSS 模块。

14 组件
4 主题色
0 依赖
100% CSS

01 · 按钮 Button

基础按钮组件，提供 5 种变体与 3 种尺寸。继承 .btn 类即可使用。
